C++ 函数

一、函数简介

  • 函数是 一组 一起执行同一个任务的语句。如何划分代码到不同的函数中由开发者决定，但在逻辑上，划分通常是根据每个函数执行一个特定的任务来进行的。

  • 每个 C++ 程序都至少有一个函数，即主函数 main()。

  • 函数 声明 告诉编译器函数的名称、返回类型和参数。

  • 函数 定义 提供了函数的实际主体。

  • C++ 标准库提供了大量程序可以调用的内置函数。例：函数 strcat() 用来连接两个字符串，函数 memcpy() 用来复制内存到另一个位置。

  • 函数还有很多叫法，比如方法、子例程或程序等。

二、函数定义

  • C++ 函数由一个 函数头 和一个 函数体 组成，定义格式如下：

    • return_type：函数的返回值类型。一个函数可以返回一个值，有些函数仅需执行函数体而不返回值，在这种情况下，return_type 可设置为关键字 void。
    • function_name：函数的实际名称，需要符合标识符命名规范。函数名和参数列表一起构成了函数签名。
    • parameter list：参数列表。包括函数参数的类型、顺序、数量。参数是可选的，也就是说，函数可能不包含参数。这里的参数就像是占位符，被称为 形式参数。
    • body of the function：函数体。包含一组定义函数执行任务的语句。

  • C++ 不允许函数嵌套定义，即函数不能在其他函数中定义。

  • 示例：定义一个 max 函数，函数有两个参数 num1 和 num2，返回这两个数中较大的数。

三、函数声明

  • 函数声明会告诉编译器函数名称及如何调用函数。函数的实际主体可以单独定义。
  • 函数声明格式：return_type function_name(parameter list);
  • 针对上面定义的函数 max() 的函数声明为：int max(int num1, int num2);
    • 在函数声明中，参数的名称并不重要，只有参数的类型是必需的，因此 int max(int, int); 也是有效的声明。
  • 在一个源文件中定义函数且在另一个文件中调用函数时，函数声明是必需的。在这种情况下，应该在调用函数的文件顶部声明函数。

四、函数调用

  • 创建 C++ 函数时，会定义函数做什么，然后通过调用函数来完成已定义的任务。
  • 函数调用格式：function_name(parameter list);
    • function_name：被调用函数的名称。
    • parameter list：当函数被调用时，需要向参数列表传递值，这些值被称为 实际参数。实际参数的数据类型与数量与函数定义要相符。
  • 当程序调用函数时，程序控制权会转移给被调用的函数。被调用的函数执行已定义的任务，当函数的返回语句被执行或到达函数的结束括号时，会把程序控制权交还给主程序。
  • 如果函数返回一个值，则可以使用一个变量存储返回值：return_type var = function_name(parameter list);。

五、函数参数

  • 如果函数要使用参数，则必须声明接受参数值的变量，这些变量被称为函数的 形式参数。
  • 形式参数就像函数内的其他局部变量，在进入函数时被创建，退出函数时被销毁。
  • 当调用函数时，有 传值调用、**指针调用 **和 引用调用 三种向函数传递参数的方式。

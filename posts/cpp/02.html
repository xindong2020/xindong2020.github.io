C++ 基本数据类型

|总字数:3.4k|阅读时长:14分钟|浏览量:

一、简介

  • C++ 规定在创建一个变量或常量时，必须明确指定相应的数据类型，否则无法为变量分配内存。

  • C++ 内置了对许多不同数据类型的支持，这些被称为基本数据类型，又被称为基本类型或内置类型。

  • 数据类型存在的意义即为变量分配适合的内存空间。

  • 下表列出了 7 种最基本的 C++ 数据类型：

    类型 关键字
    整型 int
    浮点型 float
    字符型 char
    双浮点型 double
    布尔型 bool
    无类型 void
    宽字符型 wchar_t
    • wchar_t 的定义：typedef short int wchar_t;，因此 wchar_t 实际上的空间与 short int 相同。

  • 一些基本类型可以使用一个或多个类型修饰符进行修饰，然后成为新的数据类型，下表列出了 7 种修饰符：

    修饰符 描述 示例
    signed 表示有符号类型（默认） signed int x = -10;
    unsigned 表示无符号类型 unsigned int y = 10;
    short 表示短整型 short int z = 100;
    long 表示长整型 long int a = 100000;
    const 表示常量，值不可修改 const int b = 5;
    volatile 表示变量可能被意外修改，禁止编译器优化 volatile int c = 10;
    mutable 表示类成员可以在 const 对象中修改 mutable int counter;
    • 默认情况下，int、short、long 都是带符号的，即 signed。

  • 下表显示了各种变量类型在内存中存储值时需要占用的内存，以及该类型的变量所能存储的最大值和最小值。

    数据类型 描述 大小（字节） 范围/取值示例
    bool 布尔类型，表示真或假 1 true 或 false
    char 字符类型，通常用于存储 ASCII 字符 1 -128 ~ 127
    unsigned char 无符号字符类型 1 0 ~ 255
    wchar_t 宽字符类型，用于存储 Unicode 字符 Windows 下 2 ；Linux 下 4 取决于操作系统
    char16_t 16 位 Unicode 字符类型（C++11 引入） 2 0 ~ 65,535
    char32_t 32 位 Unicode 字符类型（C++11 引入） 4 0 ~ 4,294,967,295
    short（short int） 短整型 2 -32,768 ~ 32,767
    unsigned short 无符号短整型 2 0 ~ 65,535
    int 整型 4 -2,147,483,648 ~ 2,147,483,647
    unsigned int 无符号整型 4 0 ~ 4,294,967,295
    long（long int） 长整型 Windows 下 4 ；Linux 下 4 或 8 取决于操作系统
    unsigned long 无符号长整型 Windows 下 4 ；Linux 下 4 或 8 取决于操作系统
    long long 长长整型（C++11 引入） 8 -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
    unsigned long long 无符号长长整型（C++11 引入） 8 0 ~ 18,446,744,073,709,551,615
    float 单精度浮点数 4 约 ±3.4e±38（6 ~ 7 位有效数字）
    double 双精度浮点数 8 约 ±1.7e±308（15 位有效数字）
    long double 扩展精度浮点数 8、10、12 或 16 取决于操作系统和编译器

2.2 无符号的整数类型

  • 带符号整数类型前加上关键字 unsigned 即成为无符号整数类型。
  • 无符号整数类型的取值范围宽度与带符号整数类型一致，只是将负数取值范围相应向右移动。

2.3 C++ 标准库 <climits>

  • <climits> 是 C++ 标准库中的一个头文件，提供了与整数类型相关的限制和特性。它定义了一组常量，描述了各种整数类型（如 char、int、long 等）的最小值、最大值和其他相关属性。这些常量来自 C 标准库的 <limits.h> 头文件。
  • 使用 <climits> 库需要在文件开头引入：#include<climits>

三、字符类型

  • 字符类型 char 仅占用 1 个字节，用于存储单个字符，例：字母和数字。

  • char 类型可以接收单引号 '' 包裹的 单个字符 或 -128 ~ 127 之间的数字。

  • 字符型变量并不是把字符本身存放到内存中，而是将对应的 ASCII 字符集编码存入到内存中。

  • char 类型接收数字后，当输出时会自动转成 ASCII 字符集对应的字符进行输出，负值为保留字符集，不同编译器显示不同。

四、浮点数类型

类型名称 大小（字节） 取值范围
float 4 约 ±3.4e±38（6 ~ 7 位有效数字）
double 8 约 ±1.7e±308（15 位有效数字）
long double 8、10、12 或 16 取决于操作系统和编译器

  • C++ 中无明确标注的浮点数默认是 double 类型。

  • 如果创建 float 类型数据，需要使用 f 或 F 后缀，如果不使用后缀，会发生隐式转换，即 double 类型转换为float 类型。

  • 如果创建 long double 类型数据，需要使用 l 或 L 后缀，如果不使用后缀，会发生隐式转换，即 double 类型转换为long double 类型。

  • C++ 默认输出 6 位有效数字（含小数点前数字）。

  • C++ 有两种写入浮点数的方式：

    • 第一种使用标准的小数点符号，例：3.1415926。
    • 第二种使用科学计数法表示，例：2.22E5 表示 222000。

  • 示例：

    • sizeof 是一个关键字，它是一个编译时运算符，用于判断变量或数据类型的字节大小。sizeof 运算符还可用于获取类、结构、共用体和其他用户自定义数据类型的大小。语法：sizeof (data type)
    • typeid 是一个关键字，它用来获取一个表达式的类型信息。typeid(expression).name() 可以返回表达式的数据类型。

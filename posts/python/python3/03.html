一、Python 序列

1.1 概述

  • Python 序列（Sequence）是指按 特定顺序 依次排列的一组数据。

  • 序列三要素：有元素存在，元素之间有顺序，元素可以使用下标进行索引。

  • 序列类型包括 列表（<class 'list'>）、元组（<class 'tuple'>）和 字符串（<class 'str'>）。

  • 列表和元组相似，它们都 按顺序 保存元素，每个元素都有自己的 索引，因此列表和元组的元素都可以通过索引（index）来访问。它们的 区别 在于：列表可以修改，而元组不可修改。

  • 字符串也是一种常见的序列，它也可以直接通过索引访问字符串内的字符。

  • 字典（<class 'dict'>）和 集合（<class 'set'>）不是序列类型，可以使用 collections 库中的 abc 模块里面的 Sequence 基类 来判断一个类型是不是属于序列。

1.2 序列索引

  • 序列中，每个元素都有属于自己的编号（索引）：从起始元素开始，索引值从 0 开始递增。

  • 序列索引格式：seq[index]。

  • Python 支持索引值是 负数，此类索引是 从右向左 计数：从最后一个元素开始，索引值从 -1 开始递减。

    • 负值索引的最小值为序列长度的相反数，即正值索引最大值 + 1 的相反数。

  • 序列索引超出范围时报错：IndexError: string index out of range。

1.3 序列切片

  • 切片操作是访问序列中元素的另一种方法，它可以访问一定范围内的元素，通过 切片 操作，可以 生成一个新的序列。

  • 序列实现切片操作的语法格式：seqName[start : end : step] = iterable

    • seqName：表示序列的名称；

    • start：表示切片的 开始 索引位置（包含 该位置），可以为负。可选，默认为 0，即序列开始。

    • end：表示切片的 结束 索引位置（不包含 该位置），可以为负。可选，默认为序列长度，即 len(seq)。

      • 当 end 超出序列长度时，不会报错，默认截取到序列结尾。

    • step：表示在切片过程中，隔几个存储位置（包含当前位置）取一次元素，默认为 1。可选，如果 step 的值大于 1，则在进行切片操作时，会 “跳跃式” 的取元素。

      • step 为 正 表示 从左向右，要求 start < end，否则为空。

        • 如果 start 省略，默认为 0 或 -len(seq)；

        • 如果 end 省略，则默认为 len(seq)。

      • step 为 负 表示 从右向左，要求 start > end，否则为空。

        • 如果 start 省略，默认为 -1 或 len(seq)-1；

        • 如果 end 省略，则默认为 -len(seq)-1。

    • iterable：可迭代对象，表示给序列的切片部赋予的值必须是可迭代类型。否则报错：TypeError: can only assign an iterable。

      • 注：当 iterable 为一个字符串时，字符串会被分成单个字符插入序列中。

  • 当 start 或 end 超出有效索引范围​时，切片操作不会抛出异常，而是仅保留有效索引范围的值。

  • 索引会降维（每索引一次降低一个维度），切片不会降维。

1.4 序列相加

  • Python 中，支持两种 类型相同 的序列使用 “+” 运算符做相加操作，它会将两个序列进行 连接 并生成一个新的序列，这个新的序列 不会去除重复 的元素且 不改变序列类型。

1.5 序列相乘

  • Python 中，使用数字 n 乘以一个序列会生成新的序列，其内容为原来序列被重复 n 次的结果。

  • 特殊地，列表类型在进行乘法运算时，可以实现初始化指定长度列表的功能。例：list = [None] * 5 将创建一个长度为 5的列表，列表中的每个元素都是 None，即 [None, None, None, None, None]。

1.6 检查元素是否为序列成员

  • Python 中，可以使用 in 关键字 检查某元素是否为序列成员，格式：value in sequence，其中 value 表示要检查的元素，sequence 表示指定的序列。

    • 集合和字典同样支持 in 操作符，其中字典的 in 操作符用于判断 键 是否存在于字典中，如果存在返回 true，否则返回 false。

  • not in 关键字和 in 关键字用法相同，但功能恰好相反的，它用来检查某个元素是否不包含在指定的序列中。

二、list 列表

2.1 概述

  • 类型 <class 'list'>。

  • Python 中没有数组，但是加入了更加强大的列表。

  • 列表是一种 可变数据类型。

  • 列表在内存中的 存储空间 是 连续 的，所以列表支持索引和切片，但列表中的元素 引用 的 地址 却在绝大多数情况下是 不连续 的（列表存储不同的数据类型）。

    • 列表相当于将数据的内存地址按照一定顺序排列好。

  • 列表是可变的，所以我们可以通过 索引 和 切片 的方式来对列表的元素进行修改。

  • 从 表现形式 上看，列表会将所有元素都放在一对中括号 [] 里面，相邻元素之间用逗号 , 分隔：[element1, element2, ... , elementn]，其中 element1 ~ elementn 表示列表中的元素，个数没有限制，数据类型只要是 Python 支持的就可以。

  • 从 存储内容 上看，列表可以存储数字、字符串、列表、元组等任何类型的数据，并且同一个列表中元素的类型也可以不同。

    • 在使用列表时，虽然可以将不同类型的数据放入到同一个列表中，但通常情况下不这么做，同一列表中只放入同一类型的数据，这样可以提高程序的可读性。

2.2 创建列表

  • 创建列表的方法可分为 两种：

    • 使用 [] 直接创建 列表：listname = [element1, element2, ... , elementn]，其中 listname 表示变量名，element1 ~ elementn 表示列表元素。可以创建空列表，例：emptylist = []。

    • 使用 内置函数 list([iterable]) 将 可迭代对象 转换为 list 类型，可以转换的对象有字符串，列表，元组，range 函数的返回值，字典 keys，values 方法的返回值。如果没有传入参数，则创建空列表，例：emptylist = list()。

  • 列表变量指向引用地址，该引用地址中保存有列表中每个元素所在的内存地址。

  • 注：列表中存储的数据都是数据对应的内存地址，没有真实数据。

2.3 访问列表元素

  • 列表是 Python 序列中的一种，我们可以使用 索引（index）访问 列表中的某个元素（得到的是一个元素的值），也可以使用 切片访问 列表中的一组元素（得到的是一个新的子列表）。

    • 使用 索引 访问列表元素的格式为：listname[i]，其中 listname 表示列表名字，i 表示索引值。列表的索引可以是正数，也可以是负数。

    • 使用 切片 访问列表元素的格式为：listname[start : end : step]，其中 listname 表示列表名字，start 表示起始索引（包含该位置），end 表示结束索引（不包含该位置），step 表示步长。

2.4 删除列表（del 关键字）

  • 对于已经创建的列表，如果不再使用，可以使用 del 关键字 将其删除。

  • 格式：del listname，其中 listname 表示要删除列表的名称。

  • del 语句在删除变量时，不是直接删除数据，而是 解除 变量对数据的 引用，当数据 引用计数 为 0 时，数据就变成了一个可回收的对象，然后内存会被不定期回收。

  • 注：Python 中的所有删除操作都是解除引用关系，而不是直接将数据删除。

2.5 列表方法

  • Python 列表的数据类型为 list，我们可使用 dir(list) 来查看该类型包含哪些方法。

    • 不以双下划线 __ 开头结尾的方法都是列表的对象方法，即 append, clear, copy, count, extend, index, insert, pop, remove, reverse, sort。

  • 修改列表的方法都是对原列表进行修改，即 in-place 操作（原地操作），无返回值，或者说返回值为 None。

2.5.1 列表增加元素

（1）append() 方法

  • 格式：listname.append(obj)，用于在 列表末尾 追加 元素。

    • listname：要追加元素的列表。

    • obj：要添加到列表末尾的 任意数据，可以是数字、单个元素、列表、元组、字符串等任意对象。

  • listname.append(obj) 作用等价于 listname[len(listname):] = [obj]。

  • 使用 append 方法将列表、元组等 可迭代数据 添加到列表末尾时，此方法会将它们作为一个 整体 添加到列表中，从而形成包含元素为列表或元组的新列表。

  • 示例：

（2）extend() 方法（批量 append）

  • 格式：listname.extend(iterable)，用于在 列表末尾 追加 元素。

    • listname 表示要追加可迭代对象的列表。

    • iterable 表示要添加到列表末尾的 可迭代数据，它可以是列表、元组、字符串等。

  • listname.extend(obj) 作用等价于 listname[len(listname):] = iterable。

  • extend(iterable) 和 append(obj) 的不同之处：

    • extend 的参数要求是可迭代对象，append 的参数可以是任意数据。

    • extend 不会将列表、元祖、字符串等可迭代对象视为一个整体，而是将可迭代对象中包含的元素 逐个添加 到列表中。

（3）insert() 方法

  • append 和 extend 方法只能在 列表末尾 插入元素，如果希望在 列表中间 某个位置插入元素，那么可以使用 insert 方法。

  • 格式：listname.insert(index , obj)，用于在列表指定索引处插入 一个元素。

    • index 表示要插入位置的索引。

    • obj 表示要添加到列表的任意数据。

  • 使用 insert 方法将列表、元祖等 可迭代数据 插入列表时，insert 方法也会将它们作为一个 整体 插入到列表中，这一点和 append 方法是一样的。

（4）del 关键字

  • del 是 Python 中用来执行删除操作的关键字，它不仅可以删除整个列表，还可以删除列表中的元素。

    • 使用 del 删除列表中的 单个元素，格式：del listname[index]，其中 listname 表示列表名称，index 表示元素的索引值。

    • 使用 del 删除列表中的 切片，格式：del listname[start : end ： step]，其中 start 表示起始索引，end 表示结束索引，step 表示步长。del 会删除从索引 start 到 end 之间，步长为 step 的元素，不包括 end 索引的元素。

  • del 只是解除引用关系，而不是直接将数据删除。

    • 列表的 内存自动管理 机制可以控制列表长度的伸缩。

  • 如果指定步长，Python 要求 iterable 的 元素个数 与切片获取到的 元素个数 必须相同。

    • 注：step = -1 时，Python 同样要求 iterable 的 元素个数 与切片获取到的 元素个数 必须相同，且 iterable 的元素被 反向赋值 给列表切片对应位置。

  • 如果为切片赋值空列表 []，就相当于删除一组元素。

  • 如果对空切片 [n:n] 赋值，就相当于插入一组新的元素。

  • 列表变量所指向的内存地址在列表修改后并不会发生改变。

三、tuple 元组

3.1 概述

  • 类型 <class 'tuple'>。

  • 元组（tuple）是 Python 中另一个重要的序列结构，和列表类似，元组也是由一系列按特定顺序排序的元素组成。

  • 元组和列表（list）的 区别：列表的元素是可以更改的，包括修改、删除和插入元素，所以 列表是可变序列。然而，元组一旦被创建，它的元素就不可更改了，所以 元组是不可变序列。

    • 注：元组不可变只是元组中的 引用地址不可变，但是如果引用地址所指向的元素本身数据类型是可变的，那这个元素依然是可变的。例：

  • 从 表现形式 上看，元组的所有元素都放在一对小括号 ( ) 中，相邻元素之间用逗号 , 分隔，格式：(element1, element2, ... , elementn)，其中 element1 ~ elementn 表示列表中的元素，个数没有限制，数据类型只要是 Python 支持的就可以。

  • 从 存储内容 上看，元组可以存储数字、字符串、列表、元组等任何类型的数据，并且在同一个元组中，元素的类型可以不同。

3.2 创建元组

3.2.1 使用 ( ) 创建元组

  • 通过 () 创建元组后，一般使用 = 将它赋值给某个变量，格式：tuplename = (element1, element2, ..., elementn)，其中 tuplename 表示元组名，element1 ~ elementn 表示元组的元素。

  • 在 Python 中，元组通常都是使用一对小括号将所有元素包围起来的，但 小括号不是必须的，只要将各元素用逗号隔开，Python 就会将其视为元组，这种操作被称为 封包。

  • 当创建的元组中 只有一个元素 时，该元素后面必须要加一个 逗号 ,，否则 Python 解释器不会将它视为元组，只会将它视为单个元素，数据类型为元素类型。

3.2.2 使用 tuple() 方法创建元组

  • 格式：tuple([iterable])，内置函数，用于创建一个新的元组或将其它可迭代对象转换为元组类型。

    • iterable 表示可迭代对象，包括字符串、元组、range 对象等。如果未指定，则返回空元组，即 tuple() 返回 ()。

  • 示例：

3.3 访问元组

  • 和列表一样，可以使用 索引（index）访问 元组中的某个元素（得到的是一个元素的值），也可以使用 切片访问 元组中的一组元素（得到的是一个 新 的子元组）。

    • 使用 索引访问 元组元素的格式：tuplename[i]，其中 tuplename 表示元组名字，i 表示索引值。元组的索引可以是正数，也可以是负数。

    • 使用 切片访问 元组元素的格式：tuplename[start : end : step]，其中 start 表示起始索引（包含该位置），end 表示结束索引（不包含该位置），step 表示步长。

3.4 删除元组

  • 对于已经创建的元组，如果不再使用，可以使用 del 关键字将其删除。格式为：del tuplename，其中 tuplename 表示要删除元组的名称。

    • 注：元组不支持单个元素删除。

3.5 元组对象方法

  • Python 元组的数据类型为 tuple，我们可使用 dir(tuple) 来查看该类型包含哪些方法。

    • 不以双下划线 __ 开头结尾的方法都是元组的对象方法，只有 count 和 index。

3.5.1 count() 方法

  • 格式：tuplename.count(obj)，用于统计某个元素在元组中出现的次数，其中 tuplename 代表元组名，obj 表示要统计的元素。

    • 如果 count() 返回 0，就表示元组中不存在该元素，所以 count() 也可以用来判断元组中的某个元素是否存在。

  • 示例：

3.5.2 index() 方法

  • 格式：tuplename.index(obj[, start, end])，用于查找某个元素 从左至右 在元组的指定范围（[start:end]）中 第一次出现 的位置（索引）并 返回。注：返回的索引是相对于 整个 tuplename 元组 计算的，而不是 [start:end] 的子列表。

    • start 表示搜寻范围的起始索引（包含），默认为 0。

    • end 表示搜寻范围的结束索引（不包含），默认为 len(list)。

  • 如果要查找的元素不存在，index 方法会抛出异常 ValueError: tuple.index(x): x not in tuple，建议查找前使用 count() 方法进行判断。

一、字符串

1.1 概述

  • 类型 <class 'str'>。

  • 字符串是一种 不可变 的数据类型。

  • 字符串也是一种常见的 序列，所有标准的序列操作（索引、切片、相加、相乘、in 操作、序列内置函数 等）对字符串同样使用。

  • Python 没有 字符（char）这个类型，而是用 “长度为 1 的字符串” 来表示字符的概念。

1.2 字符串拼接

  • 使用逗号 , 拼接，相邻的两个字符串之间会 自动添加空格，可以应用于字符串常量和字符串变量。示例：

  • 直接拼接（中间有无空格皆可），相邻的两个字符串之间 不会自动添加空格，仅可以应用于字符串常量。示例：

  • 使用加号 + 拼接，相邻的两个字符串之间 不会自动添加空格。示例：

1.3 字符串方法

  • Python 集合的数据类型为 str，我们可使用 dir(str) 来查看该类型包含哪些方法。

  • 因为字符串是 不可变 数据类型，所以字符串方法基本都 具有返回值，即返回一个 新字符串，原字符串并不被修改。

    • 对原字符串的操作其实是应用于原字符串的 拷贝副本，操作完成后再将拷贝副本的地址赋值给新的变量，这种操作又被称为 新建操作。

1.3.3 center() 方法

  • 格式：str.center(length[, fillchar])，返回一个在指定长度 length 居中的 新字符串。

    • 注：length 小于字符串长度则直接返回 原字符串。
    • fillchar 为左右填充的字符，默认为空格。必须是单个字符，否则报错：TypeError: The fill character must be exactly one character long。

  • 当左右填充不平衡时：

    • 原字符串长度为 奇数 时，左边填充更少。

    • 原字符串长度为 偶数 时，左边填充更多。

1.3.4 count() 方法

    • end：字符串中结束搜索的位置（不包含），默认为 len(str)。

  • 与 find 方法唯一区别：如果 str 中不包含 substr，find 返回 -1，index 则抛出一个异常。

1.3.15 isdecimal() 方法

    • True：Unicode 数字，byte 数字（单字节），全角数字（双字节）。

    • False：汉字数字，罗马数字，小数，负数，复数。

    • Error：无。

  • 因为负号（-）、小数点（.）都不属于数字，所以只有非负整数返回 True。

    • 第三个元素包含第一个匹配项后面的部分。

  • 如果字符串 value 未被找到，则返回的元组包含 原字符串 及 两个空字符串。

    • substr：搜索的子字符串。

    • start：搜索范围的开始。默认为第一个字符，第一个字符的索引为 0。

    • end：搜索范围的结束（不包含），默认为 len(str)。

    • 注：字符串并未倒置。

1.3.33 rindex() 方法

    • substr：搜索的子字符串。

    • start：搜索范围的开始。默认为第一个字符，第一个字符的索引为 0。

    • end：搜索范围的结束（不包含），默认为 len(str)。

    • 注：字符串并未倒置。

  • 与 rfind 方法唯一区别：如果 str 中不包含 substr，rfind 返回 -1，rindex 则抛出一个异常。

1.3.36 rsplit() 方法

  • 格式：str.split(sep=None, maxsplit=-1)，通过指定分隔符 sep 对字符串进行分割，返回值为从 右侧 开始 分割 后的 字符串列表（分割结果不包含分隔符）。注：返回的字符串列表顺序依然为从左向右。

    • sep 表示用于分割字符串的分隔符，默认为所有的空白符（空格、换行符、制表符等）。

      • 注：当使用默认分割符对字符串进行分割时，返回结果中的空字符串会被丢弃。例：

    • maxsplit 表示最大分隔次数，默认为 -1，即分隔所有，返回的字符串列表长度为 maxsplit + 1。

      • 如果 maxsplit 大于最大可分割数，则分隔数量以最大分割数为准。

一、推导式

  • 推导式 comprehensions（又称解析式），是 Python 的一种 独有特性，是一种可以从一个序列构建另一个新的序列的结构体。

  • 共有 3 种推导式：列表推导式、字典推导式 和 集合推导式，这三种都是 可变数据类型。

1.1 列表推导式

  • 格式：

    • [表达式 for 变量 in 列表] 
[expression for var in list]


- `if condition`：条件语句，可以过滤列表中不符合条件的值。

    • [expression for var in list 更多的 for 循环或 if 语句]。

      • 注：更多的 for 循环或 if 语句表示 for 循环嵌套或 if 语句嵌套。

1.2 字典推导式

  • 格式：

    • {key_exp: value_exp for value in iterable}

    • {key_exp: value_exp for value in iterable if condition}。

  • 注：字典推导式只能在 value 中嵌套，因为 key 不可变。

1.3 集合推导式

  • 格式：

    • {exp for var in iterable}

    • {exp for var in iterable 更多的 for 循环或 if 语句}。

  • 注：集合去重，集合推导式不能嵌套可变数据类型（集合、列表、字典）。

  • 示例：

1.4 元组推导式（生成器表达式）

  • 格式：

    • (expression for item in Sequence)
    • (expression for item in Sequence if conditional)

  • 元组推导式可以利用 range 区间、元组、列表、字典和集合等数据类型，快速生成一个满足指定需求的元组。

  • 元组推导式和列表推导式的用法也完全相同，只是元组推导式是用 () 圆括号将各部分括起来，而列表推导式使用的是中括号 []，另外元组推导式返回的结果是一个生成器对象。

二、random 模块

  • random 模块实现了各种分布的 伪随机数 生成器。

  • 随机数生成器生成的随机数都不是真正的随机数，而是 伪随机数。随机数生成器都是在一个封闭的系统内，使用 固定的算法（通常是线性同余或平方取中），通过一个 种子（通常用时钟代替）生成随机数。这意味着，如果知道了种子和已经产生的随机数，就可能获得接下来随机数序列的信息，这就是伪随机数的 可预测性。

  • 要使用 random 模块中的函数必须先导入：import random。

  • random 模块常用函数：

    • random.random() 返回 [0.0, 1.0) 范围内的 随机浮点数。

    • random.randint(a, b) 返回 [a, b] 范围内的 随机整数。

    • random.uniform(a, b) 返回 [a, b] 或 [b, a] 范围内的 随机浮点数，a 和 b 的大小关系无所谓。

    • random.choice(seq) 从 非空序列（列表、元组、字符串）seq 返回一个 随机元素。 如果 seq 为空，则抛出 IndexError 异常。

      • 原理：随机索引，通过 seq[i] 获取序列元素。

    • random.sample(population, k) 从序列或集合中随机获取 k 个不同元素，以列表形式返回。（Python3.9 版本，集合中采样已弃用）

    • random.shuffle(x) 将可变序列（列表）x 随机打乱位置，无返回值（原地操作）。

    • random.randrange([start,] stop [,step]) 等价于从 range(start, stop, step) 中随机返回一个元素。

    • random.seed ([x]) 起固定随机数的作用，x 可以是任意数字（x 可
      以理解为种子的名字）。

      • 使用 seed 函数之后的 random 语句都返回 固定 的 “随机数”。

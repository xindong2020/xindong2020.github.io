一、Python 基本数据类型

  • Python 基本数据类型一般分为 6 种：数值（Numbers）、字符串（String）、列表（List）、元组（Tuple）、字典（Dictionary）、集合（Set）。

  • Python 的六个标准数据类型中：

    • 不可变数据（3 个）：Number（数字）、String（字符串）、Tuple（元组）。

    • 可变数据（3 个）：List（列表）、Dictionary（字典）、Set（集合）。

  • Python 内置的 type(object) 函数可以用来查询变量 object 所指向的对象类型。该函数具有返回值：object 的数据类型。

1.1 Numbers 数值

  • Numbers 数字类型是 不允许改变 的，这就意味着如果改变数字数据类型的值，将 重新分配内存空间。

  • Python 中数字类型有 3 种：整型数（int）、浮点型数（float）、复数（complex）。其中，整型数又可以分为整型（int）和布尔型（bool）。

1.1.1 int 整型数

（1）int 整型

  • 类型 <class 'int'>。

  • Python 整数不分类型，或者说它只有一种类型的整数。Python 整数的 取值范围 是 无限 的，不管多大或者多小的数字。

  • 当所用数值超过计算机自身的计算能力时，Python 会自动转用高精度计算（大数计算）。

  • 在 Python 中，可以使用多种进制来表示整数：

    • 十进制形式：常见的整数形式，由 0 ~ 9 共十个数字排列组合而成。

    • 二进制形式：由 0 和 1 两个数字组成，书写时以 0b 或 0B 开头。

    • 八进制形式：由 0 ~ 7 共八个数字组成，书写时以 0o 或 0O 开头。

    • 十六进制形式：由 0 ~ 9 十个数字和 A ~ F（或 a ~ f）六个字母组成，书写时以 0x 或 0X 开头。

  • 为了提高数字的的 可读性，Python 3.x 允许使用下划线 _ 作为数字（包括整数和小数）的 分隔符。通常每隔三个数字添加一个下划线，类似于英文数字中的逗号。例：num = 1_234.123_456。

    • 不能将下划线 _ 换成逗号 ,，否则会被识别为元组。例：

（2）bool 布尔型

  • 类型 <class 'bool'>。

  • Python3 提供了 bool 类型来表示真（对）或假（错），其中 真（对）使用 True 来代表、假（错）使用 False 来代表。

  • True 和 False 是 Python 中的 关键字，要注意字母的大小写。

  • 在 Python3 中，bool 是 int 的 子类，bool.__base__ 返回的 基类 是 <class 'int'>，True 和 False 可以与数字直接相加，类似于数值 1 和 0。在实际使用中，True == 1、False == 0 会返回 True。

  • 注：+True 返回 1，+False 返回 0。

1.1.2 float 浮点型数

  • 类型 <class 'float'>。

  • Python 只有一种小数类型，就是 float。书写时必须包含一个 小数点，否则会被当作整数处理。

  • Python 浮点数允许仅保留小数点模式，例：3.。

  • Python 中的小数有两种书写形式：

    • 十进制形式：常见的浮点数形式。

    • 指数形式：写法为 aEn 或 aen，整个表达式等价于 a×10na×10^na×10n。例：2.1E5=2.1×1052.1E5 = 2.1×10^52.1E5=2.1×105。

1.1.3 complex 复数

  • 类型 <class 'complex'>。

  • 复数由 实部（real）和 虚部（imag）构成。在 Python 中，复数的虚部以 j 或 J 作为后缀，具体格式为：a + bj，其中 a 表示实部，b 表示虚部。例：12 + 0.2j。

    • 注：当表示复数时，j 或 J 前面的系数不能省略。

  • 复数（complex）是 Python 的内置类型，直接书写即可。

  • 使用 complex() 内置函数可以表示复数，例：complex(12, 0.2) 表示复数 (1 + 2j)。

  • 具有实部和虚部的复数在输出时自动在数据两侧添加括号，仅有虚部的复数则不会添加。

  • 复数（complex）具有属性 real 和 imag，可以用来获取复数的实部和虚部。例：num.real 获得 12，num.imag 获得 0.2。

  • float([x])：内置函数，将 x 转换为一个 浮点数 并 返回。如果没有指定 x，则返回 0.0，即 float() 返回 0.0。

  • chr(x)：内置函数，将一个整数转换为它对应的字符，它是 ord() 的逆函数。

  • ord(x)：内置函数，将一个字符转换为它对应的 ASCII 数值或 Unicode 数值，它是 chr() 的逆函数。

1.2 String 字符串

1.2.1 普通字符串

  • 类型 <class 'str'>。

  • 字符串 String 是 不可变 的数据类型，它是 序列。

  • python 中不存在字符类型，所有的字符都是字符串类型。

  • 若干个字符的集合 就是一个字符串（String），Python 中的字符串使用双引号 "" 或单引号 '' 创建，双引号和单引号没有区别。

  • 当字符串内容中出现 引号 时，需要对其进行特殊处理，有 两种处理方法：

    • 在引号前面添加反斜杠 \ 对引号进行 转义，将引号作为普通文本。

    • 使用 不同的引号 创建字符串，即字符串内容中出现了单引号，可以使用双引号创建字符串，反之亦然。

  • 直接输出字符串时不显示引号，只有当字符串作为 列表元素 时，打印列表会 保留引号，并默认显示 单引号。

  • 要想 换行书写 一个比较长的字符串，可以在行尾添加反斜杠 \ 对字符串内容实现换行， \ 被称为 续行符。

1.2.2 长字符串

  • Python 提供了 长字符串 的使用。所谓长字符串，就是可以直接换行书写（不用加反斜杠 \）的字符串。

  • Python 长字符串由三个双引号 """ 或者三个单引号 ''' 创建。

  • 长字符串中的换行、空格、缩进等空白符都会 原样输出。

  • 在 “双引号长字符串中放置单引号” 或 “单引号长字符串中放置双引号” 不会导致解析错误。

  • 如果长字符串 没有赋值 给任何变量，那么这个长字符串就相当于 多行注释。

1.2.3 原始字符串（Raw String）

  • 原始字符串（Raw String）就是在普通字符串或者长字符串的开头加上 r 或 R 前缀。

    • 示例：str = r"F:\anaconda\envs\py39torch\Lib\site-packages\__pycache__"

  • 如果想要在打印的字符串中出现引号，按照普通字符串的做法需要对引号进行转义（加转义字符 \），但是和普通字符串不同的是，如果在原始字符串中使用转义字符，那么用于转义的反斜杠会变成原始字符串内容的一部分。即 print(r'I\'m a great coder!') 输出结果为 I\'m a great coder!。

  • 原始字符串中的反斜杠会对字符串 结束 的引号进行转义，因此 原始字符串的结尾处不能使用反斜杠，否则字符串结尾处的引号会被转义，导致字符串不能正确结束。在 Python 中有 两种方式 解决这个问题：

    • 改用长字符串的写法，不要使用原始字符串。

    • 单独书写反斜杠。例：print(r'F:\anaconda\envs' '\\')，输出结果 F:\anaconda\envs\，这是因为 Python 会自动将 print 函数中的两个字符串拼接在一起。

  • eval(str)：用来执行一个字符串表达式，并 返回表达式的值。

二、Python 输入/输出

2.1 input() 函数：获取用户输入的字符串

  • input(prompt) 函数的 用法 为：str = input(prompt)

    • str 表示一个字符串类型的变量，input 会将控制台读取到的字符串放入 str 中。

    • prompt 表示提示信息，它会显示在控制台，如果不写 prompt，就不会有任何提示信息。

      • 注：prompt 并不会作为返回值返回给调用方。

  • input(prompt) 是 Python 的 内置函数，用于从控制台读取用户输入的内容并 返回。

    • 当程序执行到 input(prompt) 语句时，程序会 暂停 并等待用户输入，用户输入回车后程序继续运行。

  • input(prompt) 函数总是以字符串的形式来处理用户输入的内容，所以用户输入的内容可以包含任何字符。

      • 例 2：不使用 with 语句打开文件时，不会主动关闭文件，flush=False的 print 函数中的内容不会被写入。（PyCharm 中显示不出区别）

        • 注：with 语句执行结束后会自动关闭文件。

        • 在 Anaconda Prompt 中运行上述代码后打开 flush.txt 文件，发现 hello world 并未被写入文件。只有关闭文件，即 f.close() 后才会将字符串写进文件中。

        • 下面我们将 flush 设置为 True，强制刷新缓存区，它会立即将字符串写入 flush.txt 文件，而无需使用 f.close()。

        • 如果 a、b 不等于 0，则 a % b + (-a) % b = b。

        • 如果 a、b 不等于 0，则 a % (-b) + (-a) % b = 0 。

    • 由于开方是次方的逆运算，所以可以使用 ** 运算符间接实现开方运算。例：81**(1/4) 结果为 3.0。

3.2 比较运算符（关系运算符）

  • >、<、==、>=、<=、!=，运算结果返回布尔值。
  • 字符串、列表和元组比较是将每个元素进行比较。
  • 集合比较是比较子集，即判断父子集（真子集）关系。
  • 字典不支持比较。

3.4 逻辑运算符

3.4.1 逻辑运算符

  • and（逻辑与）、or（逻辑或）、not（逻辑非）：and 和 or 运算符不返回布尔值。

    • 对于 and 运算符，两边的值 都为真 时最终结果才为 真，只要其中 有一个值为假，那么最终结果就是 假，所以 Python 按照下面的规则执行 and 运算：

      • 如果左边表达式的值为假，不计算右边表达式（短路），此时 and 会把左边 表达式的值 作为 最终结果，而 不是返回布尔值。

      • 如果左边表达式的值为真，and 会继续计算右边表达式的值，并将右边 表达式的值 作为 最终结果，而 不是返回布尔值。

    • 对于 or 运算符，两边的值 都为假 时最终结果才为 假，只要其中 有一个值为真，那么最终结果就是 真，所以 Python 按照下面的规则执行 or 运算：

      • 如果左边表达式的值为真，不计算右边表达式（短路），此时 or 会把左边 表达式的值 作为 最终结果，而不是返回布尔值。

      • 如果左边表达式的值为假，or 会继续计算右边表达式的值，并将右边 表达式的值 作为 最终结果，而 不是返回布尔值。

    • 对于 not 运算符，表达式的值为假，返回 True；表达式的值为真，返回 False。

  • 注：and 和 or 运算符具有 短路机制，即无论右侧的表达式是否报错，都会先计算左侧表达式结果，然后再根据结果决定是否执行右侧表达式。例：

    • print("Hello world" or 2 / 0) 会输出 Hello world 而不是报错。
    • print("Hello world" and 2 / 0) 会报错，因为 Hello world 为 True 会继续执行 2 / 0 而报错。

  • None、False、0、空字符串、空列表、空元组、空字典、空集合都会被认定为 False。

3.7 位运算符

  • 按位运算符需要将数字转换为 补码，然后参与运算。

    • 原码：整数转为二进制得到的就是原码。

    • 反码：除了符号位，其他位全部取反，即 0 变 1，1 变 0。

    • 补码：反码加 1。

3.7.1 位运算符（二进制）

  • &：按位与运算符，如果两个相应位都为 1，则该位的结果为 1，否则为 0。

  • |：按位或运算符，只要两个相应位有一个为 1 时，结果位就为 1。

  • ^：按位异或运算符，只要两个相应位不同，结果就为 1。

  • ~：按位取反运算符，对数据的每个二进制位取反（包含符号位），即把 1 变为 0，把 0 变为 1。规律：~x = -(x+1)。

  • <<：按位左移运算符，运算数的各二进位全部左移若干位，右边补 0。规律：x << n = x * (2 ** n)。

  • >>：按位右移运算符，运算数的各二进位全部右移若干位，左边填充对应的符号位值，即正数补 0，负数补 1，这种填充方式被称为 符号扩展。规律：x >> n = x // (2 ** n)。

    • 负数如果一直右移，最终二进制位会全部变成 1，即十进制的 -1 ( 二进制全 1 在补码中表示 -1)。

    • 补位在转回十进制的过程中不做数值。

    • 示例：

3.7.2 进制转换函数

  • bin(x)：内置函数，将一个整数转换为一个前缀为 0b 的 二进制 字符串。

  • oct(x)：内置函数，将一个整数转换为一个前缀为 0o 的 八进制 字符串。

  • hex(x)：内置函数，将一个整数转换为一个前缀为 0x 的 十六进制 字符串。

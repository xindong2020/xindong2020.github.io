一、顺序结构

  • 顺序结构就是让程序按照从头到尾的顺序依次执行每一条 Python 代码，不重复执行任何代码，也不跳过任何代码。

二、分支结构

  • 分支结构就是构造合适的分支条件和分析程序流程，根据分析程序流程选择适当的分支语句。

  • 在 Python 中，可以使用 if else 语句对条件进行判断，然后根据不同的结果执行不同的代码，这被称为 选择结构 或 分支结构。

2.1 if else 语句的语法格式

2.1.1 语法格式

  • Python 中的 if else 语句的语法格式：

    • 如果 condition_1 为 True，将执行 statement_block_1 代码块，执行完成后 跳出 if else 代码块；否则将向后运行到 elif 判断 “condition_2”。

    • 如果 condition_2 为 True，将执行 statement_block_2 代码块，执行完成后 跳出 if else 代码块；否则将向后运行到 elif 判断 “condition_3”。

    • 以此类推直到 elif condition_n。

    • 如果 condition_n 为 True，将执行 statement_block_n 代码块，执行完成后 跳出 if else 代码块。

    • 如果 “condition_n” 为 False，将向后运行到 else 语句，执行 “statement_block” 代码块。

2.1.2 注意事项

  • 表达式（condition）可以是一个单一的值或变量，也可以是由运算符组成的复杂语句，形式不限，只要它的结果能执行布尔判定即可。

    • 对于数字，Python 会把 0 和 0.0 当做 False，把其它值当做 True。

    • 对于其它类型，当值为空或 None 时，Python 会当做 False，其它值当做 “True”。

  • if、elif、else 语句的最后要使用 冒号 :，表示接下来是满足条件后要执行的 代码块。

    • 代码块 由具有 相同缩进量 的若干条语句组成。

      • Python 是以缩进来标记代码块的，代码块一定要有缩进。

      • 同一个代码块的缩进量要相同，缩进量不同的代码不属于同一个代码块。

  • 一旦 if 或 elif 后面的表达式（condition）成立，Python 就会执行它后面对应的代码块；如果所有表达式都不成立，就执行 else 后面的代码块；如果没有 else 部分，就什么也不执行。

  • 每个分支结构最多只会满足一次结果，即只会有一个 if 语句被执行，或只有 else 被执行，或都不被执行。

  • 如果 if 语句结构体中只有一条语句，可以将该语句与 if 语句写在同一行。不建议 这么书写，不符合 PEP 8 规范。

  • Python 中没有引入 ? : 这个三目运算符，而是使用已有的 if else 语句来实现相同的功能。

三、循环结构

3.1 while 循环

  • 未知循环次数，但是已知循环的停止条件，常用 while 循环。

3.1.1 while 循环

  • while 循环的语法格式如下：

  • while 循环的执行流程：

    • 初始化循环变量。

    • 判断条件表达式的值，值为真（True）时执行代码块中的语句，否则跳出循环。

    • 当循环体执行完毕后，重新判断条件表达式的值是否为真，若仍为真，则再次执行代码块。

      • 循环变量实现自增或自减。

    • 重复上述操作，直到条件表达式的值为假（False），终止循环。

  • 示例：输出 1 ~ 100 之间带有 6 和 8 的整数。

  • 如果 while 循环体中只有一条语句，可以将该语句与 while 语句写在同一行。不建议 这么书写，不符合 PEP 8 规范。

    • 当程序中不使用 <variable>，而我们又不想为变量命名时，我们可以使用下划线 _ 来接受遍历出的变量。

3.2.2 for 循环的内部机制

  • 先判断对象是否为可迭代对象，即是否存在 __iter__ 方法，如果存在则调用__iter__ 方法，返回一个 迭代器；否则，直接抛出 TypeError 异常。

  • 不断地调用迭代器的 __next__ 方法，每次调用按顺序迭代获取当前的值。

  • 迭代完所有元素，就抛出异常 StopIteration，这个异常 python 解释器自己会处理。

  • 如果使用 for 循环直接遍历字典，则迭代变量 variable 会先后被赋值为每个键值对中的 键，和遍历字典 keys() 方法的返回值是相同的。

（3）range 对象

  • range 对象是 Python 内置的对象，它可以认为是一个 不可变的数字序列，通常用于在 for 循环中指定循环的次数。

  • 格式：range([start,] stop[, step])，返回 一个从 start（默认 0，包含）开始到 stop（不包含）结束，步长为 step（默认 1）的 range 对象。

    • range 对象是一个不可变的 序列，是一个可迭代对象 iterable。

    • range 类型相比常规 list 或 tuple 的优势在于一个 range 对象总是占用固定数量的（较小）内存，不论其所表示的范围有多大，它只保存 start, stop 和 step 值。

  • range 虽然被称为函数，但 range 实际上是一个不可变的序列类型。

3.3 循环控制语句

  • Python 提供了循环控制语句（break、continue）配合循环结构的使用。

      • 当 break 用于嵌套循环时，break 会 跳出 当前循环，然后执行 外层循环 的剩余代码或继续执行 外层循环 的下一轮循环。

    • continue 语句用来告诉 Python 跳过 当前循环的剩余语句，然后继续执行 当前循环 的下一轮循环。

魔术方法

1. 概述

  • 在 Python 中，魔术方法（Magic Method）是一种 官方定义好，以双下划线 __ 包起来的 特殊方法。普通方法需要调用，而魔术方法在特定条件下可以 自动触发。

  • 程序开发人员可以对魔术方法 重写，定制自己需要的功能。

  • Python 中常用的运算符、for循环、以及类操作等都是运行在魔术方法之上的。

2. 常用魔术方法

2.1 __new__

  • 格式：__new__(cls, [...])，是在一个对象实例化的时候所调用的第一个方法，用于实例化对象。它的第一个参数是当前类，其他的参数是用来直接传递给 __init__ 方法。

  • 触发条件：实例化对象时自动触发，即 instance = MyClass(arg1, arg2)。

  • 返回值：实例对象。

2.2 __init__

  • 格式：__init__(self, [...])，此方法为类的初始化方法。当构造函数被调用的时候的任何参数都将会传给它。

  • 触发条件：实例化对象后，初始化对象时自动触发，即 instance = MyClass(arg1, arg2)。

  • 返回值：无。

  • 示例：

2.3 __call__

  • 格式：__call__(self, [...])，该方法允许一个类的实例像函数一样被调用。

  • 触发条件：把对象当作函数调用的时候自动触发，即 self(args)。

  • 返回值：根据需求决定。

  • 示例：

2.4 __getitem__

  • 格式：__getitem__(self, item)，该方法返回索引或键 item 对应的值。

  • 触发条件：使用索引或键访问元素时自动触发，即 self[item]。

  • 返回值：item 对应的元素值。

  • 示例：

2.5 __len__

  • 格式：__len__(self)，该方法用于检测实例对象的长度。

  • 触发条件：使用 len(self) 时自动触发。

  • 返回值：必须是整数类型。

  • 示例：

2.6 __str__

  • 格式：__str__(self)，该方法用于将实例对象以字符串形式显示。

  • 触发条件：打印实例对象名，格式化字符串或 str(self) 时自动触发。

  • 返回值：必须是字符串类型。

  • 示例：

2.7 __repr__

  • 格式：__repr__(self)，该方法用于将实例对象以字符串形式显示。

  • 触发条件：使用 repr(self) 时自动触发。

  • 返回值：必须是字符串类型。

2.8 __add__/__radd__

  • 格式：__add__(self, other)，实现了实例对象的加法操作。__radd__(self, other) ，实现了实例对象的反加法操作。大多数情况下，反运算的结果是与普通运算相同的。

  • 触发条件：当加法运算符 + 左侧是当前类的实例对象时自动触发 __add__。当加法运算符 + 左侧不是当前类的实例对象且右侧是的时候自动触发 __radd__

  • 返回值：加法运算后的值。

  • 示例：

2.9 __sub__/__rsub__

  • 实例对象进行减法操作时会调用该方法，格式、触发条件、返回值与加法类似。

2.10 __mul__/__rmul__

  • 实例对象进行乘法操作时会调用该方法，格式、触发条件、返回值与加法类似。

2.11 __truediv__/__rtruediv__

  • 实例对象进行除法操作时会调用该方法，格式、触发条件、返回值与加法类似。

2.12 __neg__

  • 格式：__neg__(self)，实现了实例对象的相反数操作。

  • 触发条件：当 -self 时自动触发 __neg__。

  • 返回值：相反数运算后的值。

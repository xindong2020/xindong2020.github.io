十九、大图查看模式

  • medium-zoom 或 fancybox 只能开启一个，推荐 fancybox。

  • 如果不想为某张图片添加大图查看模式，可以在 img 标签中添加 class 属性，值为 no-lightbox。例：<img src="xxxx.jpg" class="no-lightbox">。

一、菜单设置

  • butterfly 的默认菜单栏

  • 菜单栏中的每一项都指向单独的子页面，对应于 soure 中的一个文件夹（不包含 _posts 和 _draft 文件夹）。

    • 可以使用 hexo new page <title> 指令创建页面，例：hexo new page about 创建 About 界面，同时在 soure 中自动生成一个 about 文件夹。

    • About、Link、List 等属于普通页面，直接创建即可。但是 Archives（归档）、Tags（标签）、Categories（分类）等属于特殊页面，需要其他操作。

      • 归档页面（archives）是内建的，不需要手动创建。

      • 标签页面（tags）在使用 hexo new page tags 创建后，需要在 index.md 文件开头加上 type: "tags" 和 layout: "tags"，内容留空。

      • 分类页面（categories）采用相同操作。

二、图标相关

  • Butterfly 主题默认支持 font-awesome 图标，但是 font-awesome 对于国内网站的支持不足，建议使用阿里巴巴 iconfont。

阿里巴巴 iconfont

  • 阿里巴巴矢量图标管理、交流平台：https://www.iconfont.cn/

  • 找到自己需要的图标之后点击添加入库。

  • 选择购物车，然后添加至项目。

  • 如果不存在已有项目，需要创建新项目。

  • 创建新项目后，选择 Font class，查看在线链接。

    • 如果暂无代码，需要手动生成。
    • 进入生成的链接，并复制生成的代码（点击复制代码只是复制了链接地址而不是代码本身）。

  • 在 butterfly 主题文件夹（\themes\butterfly）下的 \source\css 文价夹中新建 font.css 文件，将复制的代码粘贴进去。

  • 在 butterfly 主题文件 _config.butterfly.yml 中引入 font.css 文件。

  • 在 _config.butterfly.yml 文件中使用格式 iconfont icon名 跳转链接 即可使用 iconfont 图标，其中 icon 名可以在 iconfont 项目中复制。

  • code_word_wrap：代码框中的代码是否自动换行。

一、IntelliJ IDEA 的安装与激活

1.1 IntelliJ IDEA 的安装

（1）Windows

  • 官网：https://www.jetbrains.com/idea/download/other.html 下载 Version 2021.3.3 的 2021.3.3 - Windows x64 (exe) 版本。

  • 双击安装。

（2）Ubuntu

  • 官网：https://www.jetbrains.com/idea/download/other.html 下载 Version 2021.3.3 的 2021.3.3 - Linux x86_64 (tar.gz) 版本。

  • 解压到指定文件夹：sudo tar -zxvf ideaIU-2021.3.3.tar.gz -C /opt，重命名解压后的文件夹为 idea-IU：sudo mv /opt/idea-IU-213.7172.25/ /opt/idea-IU/。

  • 在 Ubuntu 20.04 的 /usr/share/applications 文件夹中创建 idea.desktop 文件：sudo gedit /usr/share/applications/idea.desktop，然后输入下列内容，其中 Exec 与 icon 内容根据实际情况填写。

  • 如果想要在桌面生成快捷方式，只需在桌面创建 idea.desktop 文件：gedit ~/Desktop/idea.desktop，然后输入下列内容，其中 Exec 与 icon 内容根据实际情况填写。

    • 文件 idea.desktop 上右键，选择 Allow Launching。

1.2 IntelliJ IDEA 的激活

  • 运行 IntelliJ IDEA，出现 DATA SHARING 页面，自愿选择发送与否。

  • 先不导入之前的 IntelliJ IDEA 配置，点击 OK。

  • 启动页面

  • Intellij IDEA 运行成功后，会弹出下面的对话框，提示我们需要先登录 JetBrains 账户才能使用，这里我们先不管，先点击 Exit 退出，准备开始运行激活脚本。

  • 开始激活

    • 自行下载 JetBrains 2022 最新全家桶激活（10.19） 激活文件夹，然后解压。

    • 将上图所示的激活文件夹 方式3：永久激活补丁+脚本（适合最新版本，可显示到2025年）/jetbra 复制电脑某个位置。

    • 进入到 jetbra 中的 scripts 文件夹，运行脚本安装破解补丁。

      • Windows 系统，点击运行 install-current-user.vbs 脚本，为当前用户安装破解补丁。

      • Mac / Linux 系统，运行 bash install.sh 指令。

    • 启动 Intellij IDEA，选择 Activation code，填入激活码。

    • 点击 Activate 进行激活。

二、IntelliJ IDEA 的详细配置

2.1 进入设置界面

2.1.1 方式一

  • 欢迎界面下，点击 Customize $\rightarrow$ All settings... 进入设置界面。

2.1.2 方式二

  • 项目界面下，点击菜单 File $\rightarrow$ Settings... 进入设置界面。

2.2 关闭 Intellij IDEA 自动更新

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ System Settings $\rightarrow$ Updates，取消 Check IDE updates for: 和 Show What's New in the editor after an IDE update 的勾选 $\rightarrow$ OK。

2.3 配置项目默认路径

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ System Settings $\rightarrow$ Default project directory 选择项目默认路径。

2.4 创建项目

2.4.1 方式一

  • 欢迎界面 下，点击 Projects $\rightarrow$ New Project。

  • 创建 Empty Project 空项目 JavaSE $\rightarrow$ Finish

  • 创建 Module：项目上右键 $\rightarrow$ New $\rightarrow$ Module...

    • 设置 JDK：在新窗口中选择 Java $\rightarrow$ Module SDK $\rightarrow$ Add JDK...，选择 JDK 所在路径 $\rightarrow$ OK $\rightarrow$ Next。

    • 在新窗口中设置 Module name $\rightarrow$ Finish。

  • 创建包 Package：在新建 Module 的 src 文件夹上右键 $\rightarrow$ New $\rightarrow$ Package

  • 创建类 Class：在新建的包 Package 上右键 $\rightarrow$ New $\rightarrow$ Java Class

2.4.2 方式二

  • 项目界面 下，点击菜单 File $\rightarrow$ New $\rightarrow$ Projects...，然后按照上述步骤进行创建。

2.5 配置 JDK

  • File $\rightarrow$ Project Structure...

  • Project $\rightarrow$ SDK $\rightarrow$ 选择环境变量中配置好的JDK $\rightarrow$ Apply

2.6 统一编译器和编译版本

  • 进入 设置界面 $\rightarrow$ Build, Execution, Deployment $\rightarrow$ Compiler $\rightarrow$ Java Compiler $\rightarrow$ Use Compiler: 选择 javac $\rightarrow$ Project bytecode version: 根据当前项目使用的 JDK 版本进行统一 $\rightarrow$ Apply

2.7 设置主题

2.7.1 安装主题

  • 下载网址：http://www.themesmap.com/theme.html?t=down

  • 下载主题：Relax Your Eyes Green

  • 导入主题

    • 打开 设置界面 $\rightarrow$ Editor $\rightarrow$ Color Scheme $\rightarrow$ 齿轮设置 $\rightarrow$ Import Scheme $\rightarrow$ Intellij IDEA color scheme(.icis) or settings(.jar) $\rightarrow$ 选择 Relax Your Eyes Green 主题所在路径 $\rightarrow$ OK $\rightarrow$ Apply

2.7.2 设置主题

（1）方式一

  • 欢迎界面点击 Customize $\rightarrow$ Color theme，选择要设置的主题。

（2）方式二

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ Appearance $\rightarrow$ Theme，选择要设置的主题。

2.8 设置字体

2.8.1 IDEA 界面字体

（1）方法一

  • 在欢迎界面下，点击 Customize $\rightarrow$ 在 Accessibility 的 IDE font 中进行设置 IDEA 界面字体大小。

（2）方法二
  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ Appearance $\rightarrow$ 设置 IDEA 界面字体大小。

2.8.2 代码编辑界面字体

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ Font $\rightarrow$ 设置代码编辑界面字体大小及行距。

2.8.3 Ctrl + 鼠标滚轮修改字体大小

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ 勾选 Change font size with Ctrl+Mouse Wheel in: $\rightarrow$ 自选 Active editor 或 All editors

2.8.4 控制台字体

  • 默认情况下无需调整，会跟随代码编辑界面使用一致的设置，如需自定义则可以进入设置界面 $\rightarrow$ Editor $\rightarrow$ Color Scheme $\rightarrow$ Console Font $\rightarrow$ 勾选 Use console font instead of the default，然后自定义控制台字体

2.9 行号和方法分隔符

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Appearance $\rightarrow$ 勾选 Show line numbers 显示行号，勾选 Show method separators 显示方法分隔符

2.10 忽略大小写匹配

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Code Completion $\rightarrow$ 取消勾选 Match case

2.11 配置自动导包

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Auto Import $\rightarrow$ Insert imports on paste: 选择 Always $\rightarrow$ 勾选 Add unambiguous imports on the fly 和 Optimize imports on the fly

2.12 配置编码

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ File Encodings $\rightarrow$ Global Encodings: 和 Project Encodings: 选择 UTF-8 $\rightarrow$ Default encoding for properties files: 选择 UTF-8 $\rightarrow$ 勾选 Transparent native-to-ascll conversion $\rightarrow$ Create UTF-8 files: 选择 with NO BOM $\rightarrow$ Apply

三、IntelliJ IDEA 快捷键

3.1 Ctrl

快捷键 说明
Ctrl + F 在当前文件进行文本查找（必备）
Ctrl + R 在当前文件进行文本替换（必备）
Ctrl + Z 撤销（必备）
Ctrl + C 复制光标所在行 或 复制选择内容（必备）
Ctrl + D 复制光标所在行并插入到下一行 或 复制选择内容并插入到右侧（必备）
Ctrl + X 剪切光标所在行 或 剪切选择内容（必备）
Ctrl + Y 删除光标所在行 或 删除选中的行（必备）
Ctrl + W 递进式选择代码块。可选中光标所在的单词或段落，连续按会在原有选中的基础上再扩展选中范围（必备）
Ctrl + E 显示最近打开的文件记录列表（必备）
Ctrl + N 根据输入的 类名 查找类文件（必备）
Ctrl + G 在当前文件跳转到指定行处
Ctrl + J 插入自定义动态代码模板 （必备）
Ctrl + P 方法参数提示显示 （必备）
Ctrl + Q 光标所在的变量 / 类名 / 方法名等上面（也可以在提示补充的时候按），显示文档内容
Ctrl + U 前往当前光标所在的方法的父类的方法 / 接口定义 （必备）
Ctrl + B 进入光标所在的方法/变量的接口或是定义处，等效于 Ctrl + 左键单击 （必备）
Ctrl + K 版本控制 提交 项目，需要此项目有加入到版本控制才可用
Ctrl + T 版本控制 更新 项目，需要此项目有加入到版本控制才可用
Ctrl + H 显示当前类的层次结构
Ctrl + O 选择可重写的方法
Ctrl + I 选择可继承的方法
Ctrl + + 展开代码
Ctrl + - 折叠代码
Ctrl + / 注释光标所在行代码，会根据当前不同文件类型使用不同的注释符号 （必备）
Ctrl + [ 移动光标到当前所在代码的花括号开始位置
Ctrl + ] 移动光标到当前所在代码的花括号结束位置
Ctrl + F1 在光标所在的错误代码处显示错误信息 （必备）
Ctrl + F3 调转到所选中的词的下一个引用位置 （必备）
Ctrl + F4 关闭当前编辑文件
Ctrl + F8 在 Debug 模式下，设置光标当前行为断点，如果当前已经是断点则去掉断点
Ctrl + F9 执行 Make Project 操作
Ctrl + F11 选中文件 / 文件夹，使用助记符设定 / 取消书签 （必备）
Ctrl + F12 弹出当前文件结构层，可以在弹出的层上直接输入，进行筛选
Ctrl + Tab 编辑窗口切换，如果在切换的过程又加按上delete，则是关闭对应选中的窗口
Ctrl + End 跳到文件尾
Ctrl + Home 跳到文件头
Ctrl + Space 基础代码补全，默认在 Windows 系统上被输入法占用，需要进行修改，建议修改为 Ctrl + 逗号 （必备）
Ctrl + Delete 删除光标后面的单词或是中文句 （必备）
Ctrl + BackSpace 删除光标前面的单词或是中文句 （必备）
Ctrl + 1,2,3...9 定位到对应数值的书签位置 （必备）
Ctrl + 左键单击 在打开的文件标题上，弹出该文件路径 （必备）
Ctrl + 光标定位 按 Ctrl 不要松开，会显示光标所在的类信息摘要
Ctrl + $\leftarrow$ 光标跳转到当前单词 / 中文句的左侧开头位置 （必备）
Ctrl + $\rightarrow$ 光标跳转到当前单词 / 中文句的右侧开头位置 （必备）
Ctrl + $\uparrow$ 等效于鼠标滚轮向前效果 （必备）
Ctrl + $\downarrow$ 等效于鼠标滚轮向后效果 （必备）

3.2 Alt

快捷键 说明
Alt + ` 显示版本控制常用操作菜单弹出层（必备）
Alt + Q 弹出一个提示，显示当前类的声明 / 上下文信息
Alt + F1 显示当前文件选择目标弹出层，弹出层中有很多目标可以进行选择 （必备）
Alt + F2 对于前面页面，显示各类浏览器打开目标选择弹出层
Alt + F3 选中文本，逐个往下查找相同文本，并高亮显示
Alt + F7 查找光标所在的方法 / 变量 / 类被调用的地方
Alt + F8 在 Debug 的状态下，选中对象，弹出可输入计算表达式调试框，查看该输入内容的调试结果
Alt + Home 定位 / 显示到当前文件的 Navigation Bar
Alt + Enter IntelliJ IDEA 根据光标所在问题，提供快速修复选择，光标放在的位置不同提示的结果也不同 （必备）
Alt + Insert 代码自动生成，如生成对象的 set / get 方法，构造函数，toString() 等 （必备）
Alt + $\leftarrow$ 切换当前已打开的窗口中的子视图，比如Debug窗口中有Output、Debugger等子视图，用此快捷键就可以在子视图中切换 （必备）
Alt + $\rightarrow$ 按切换当前已打开的窗口中的子视图，比如Debug窗口中有Output、Debugger等子视图，用此快捷键就可以在子视图中切换 （必备）
Alt + $\uparrow$ 当前光标跳转到当前文件的前一个方法名位置 （必备）
Alt + $\downarrow$ 当前光标跳转到当前文件的后一个方法名位置 （必备）
Alt + 1,2,3...9 显示对应数值的选项卡，其中 1 是 Project 用得最多 （必备）

3.3 Shift

快捷键 说明
Shift + F1 如果有外部文档可以连接外部文档
Shift + F2 跳转到上一个高亮错误 或 警告位置
Shift + F3 在查找模式下，查找匹配上一个

Shift + F8 在 Debug 模式下，跳出，表现出来的效果跟 F9 一样
Shift + F9 等效于点击工具栏的 Debug 按钮
Shift + F10 等效于点击工具栏的 Run 按钮
Shift + F11 弹出书签显示层 （必备）
Shift + Tab 取消缩进 （必备）
Shift + ESC 隐藏当前 或 最后一个激活的工具窗口
Shift + End 选中光标到当前行尾位置
Shift + Home 选中光标到当前行头位置
Shift + Enter 开始新一行。光标所在行下空出一行，光标定位到新行位置 （必备）
Shift + 左键单击 在打开的文件名上按此快捷键，可以关闭当前打开文件 （必备）
Shift + 滚轮前后滚动 当前文件的横向滚动轴滚动 （必备）

Ctrl + Alt + Enter 光标所在行上空出一行，光标定位到新行 （必备）

快捷键 说明
Alt + Shift + N 选择 / 添加 task （必备）
Alt + Shift + F 显示添加到收藏夹弹出层 / 添加到收藏夹
Alt + Shift + C 查看最近操作项目的变化情况列表
Alt + Shift + I 查看项目当前文件
Alt + Shift + F7 在 Debug 模式下，下一步，进入当前方法体内，如果方法体还有方法，则会进入该内嵌的方法中，依此循环进入
Alt + Shift + F9 弹出 Debug 的可选择菜单
Alt + Shift + F10 弹出 Run 的可选择菜单
Alt + Shift + 左键双击 选择被双击的单词 / 中文句，按住不放，可以同时选择其他单词 / 中文句 （必备）
Alt + Shift + $\uparrow$ 向上移动光标所在行（必备）
Alt + Shift + $\downarrow$ 向下移动光标所在行（必备）

3.7 Ctrl + Shift + Alt

快捷键 说明
Ctrl + Shift + Alt + V 无格式黏贴 （必备）
Ctrl + Shift + Alt + N 前往指定的变量 / 方法
Ctrl + Shift + Alt + S 打开当前项目设置 （必备）
Ctrl + Shift + Alt + C 复制参考信息

3.8 其他

快捷键 说明
F2 跳转到下一个高亮错误 或 警告位置 （必备）
F3 在查找模式下，定位到下一个匹配处
F4 编辑源 （必备）
F7 在 Debug 模式下，进入下一步，如果当前行断点是一个方法，则进入当前方法体内，如果该方法体还有方法，则不会进入该内嵌的方法中
F8 在 Debug 模式下，进入下一步，如果当前行断点是一个方法，则不进入当前方法体内
F9 在 Debug 模式下，恢复程序运行，但是如果该断点下面代码还有断点则停在下一个断点上
F11 添加书签 （必备）
F12 回到前一个工具窗口 （必备）
Tab 缩进 （必备）
ESC 从工具窗口进入代码文件窗口 （必备）
连按两次 Shift 弹出 Search Everywhere 弹出层

三、IntelliJ IDEA 的其他操作

3.1 删除类文件

  • 文件名上右键 $\rightarrow$ Delete...

3.2 修改类名称

  • 文件名上右键 $\rightarrow$ Refactor $\rightarrow$ Rename $\rightarrow$ 改名 $\rightarrow$ Refactor

3.3 修改模块名称

  • 模块名上右键 $\rightarrow$ Refactor $\rightarrow$ Rename $\rightarrow$ Rename module and directory $\rightarrow$ 改名 $\rightarrow$ Refactor

3.4 导入模块

  • File $\rightarrow$ New $\rightarrow$ Module from Existing Sources... $\rightarrow$ 填写模块所在路径 $\rightarrow$ OK

3.5 删除模块

3.5.1 项目中移除

  • 模块名上右键 $\rightarrow$ Remove Module $\rightarrow$ Remove

3.5.2 彻底删除

  • 在项目中移除模块 $\rightarrow$ 再次在模块名上右键 $\rightarrow$ Delete... $\rightarrow$ Delete

3.6 打开工程

  • 菜单栏 File $\rightarrow$ Open $\rightarrow$ 找到工程所在路径 $\rightarrow$ OK

  • 菜单栏 File $\rightarrow$ Close Project

3.7 关闭工程

  • 菜单栏 File $\rightarrow$ Close Project

一、PyCharm 的安装与激活

1.1 PyCharm 的安装

（1）Windows

  • 官网：https://www.jetbrains.com/pycharm/download/other.html 下载 Version 2022.3 的 2022.3 - Windows (exe) 版本。

  • 双击 pycharm-professional-2022.3.exe 安装包进行安装：

    • 建议勾选 Create Destop Shortcut 和 Create Associations。

（2）Ubuntu

  • 官网：https://www.jetbrains.com/pycharm/download/other.html 下载 Version 2022.3 的 2022.3.3 - Linux (tar.gz) 版本。

  • 安装

  • 创建一个桌面条目，方便从 “活动” 菜单启动 PyCharm：sudo vim /usr/share/applications/pycharm.desktop

    • 加入下列内容：

1.2 PyCharm 的激活

  • 运行 PyCharm，出现 “PyCharm User Agreement”，勾选 “I confirm that I have read and accept the terms of this User Agreement” $\rightarrow$ Continue，然后自愿选择是否进行数据分享。

  • 出现 Import PyCharm Settings 页面，不导入之前的 PyCharm 配置，勾选 Do not import settings，点击 OK。

  • 启动页面

  • PyCharm 运行成功后，会弹出下面的对话框，提示我们需要先登录 JetBrains 账户才能使用，这里我们先不管，先点击 Exit 退出，准备开始运行激活脚本。

  • 开始激活

    • 自行下载 JetBrains 2022 最新全家桶激活（10.19） 激活文件夹，然后解压。

    • 将上图所示的激活文件夹 方式3：永久激活补丁+脚本（适合最新版本，可显示到2025年）/jetbra 复制电脑某个位置。

    • 进入 /jetbra/scripts 文件夹，双击执行 install-current-user.vbs 破解脚本。

    • 执行 install-current-user.vbs 脚本会自动在环境变量 $\rightarrow$ 用户变量下添加 APPCODE_VM_OPTIONS、CLION_VM_OPTIONS、DATAGRIP_VM_OPTIONS、DATASPELL_VM_OPTIONS、DEVECOSTUDIO_VM_OPTIONS、GATEWAY_VM_OPTIONS、GOLAND_VM_OPTIONS、IDEA_VM_OPTIONS、JETBRAINS_CLIENT_VM_OPTIONS、JETBRAINSCLIENT_VM_OPTIONS、PHPSTORM_VM_OPTIONS、PYCHARM_VM_OPTIONS、RIDER_VM_OPTIONS、RUBYMINE_VM_OPTIONS、STUDIO_VM_OPTIONS、WEBIDE_VM_OPTIONS 和 WEBSTORM_VM_OPTIONS 变量，并且会自动在这些环境变量的对应文件末尾引入破解补丁 -javaagent:E:\Pycharm\Activate\jetbra\ja-netfilter.jar=jetbrains。

    • 启动 PyCharm，选择 Activation code，填入激活码。

    • 点击 Activate 进行激活后，可以看到激活到 2025 年。

二、PyCharm 的详细配置

2.1 进入设置界面

2.1.1 方式一

  • 欢迎界面下，点击 Customize $\rightarrow$ All settings... 进入设置界面。

2.1.2 方式二

  • 项目界面下，点击菜单 File $\rightarrow$ Settings... 进入设置界面。

2.2 关闭 PyCharm 自动更新

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ System Settings $\rightarrow$ Updates，取消 Check IDE updates for:Stable Releases 的勾选 $\rightarrow$ Apply。

2.3 配置项目默认路径

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ System Settings $\rightarrow$ Default project directory 选择项目默认路径。

2.4 配置 Python 解释器

  • 进入设置界面 $\rightarrow$ Python Interpreter $\rightarrow$ Add Interpreter $\rightarrow$ Add Local Interpreter... $\rightarrow$ Add Local Interpreter...

2.5 创建项目

2.5.1 方式一

  • 欢迎界面下，点击 Projects $\rightarrow$ New Project。然后创建 Pure Python 项目，Location 位置填写项目路径，最后的文件夹即为项目名：pythonProject。选择 Previously configured interpreter $\rightarrow$ 配置 Python 编译器所在路径 $\rightarrow$ Create

2.5.2 方式二

  • 项目界面 下，点击菜单 File $\rightarrow$ New Project...，然后按照上述步骤进行创建。

2.6 配置编码

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ File Encodings $\rightarrow$ Global Encodings: 和 Project Encodings: 选择 UTF-8 $\rightarrow$ Default encoding for properties files: 选择 UTF-8 $\rightarrow$ 勾选 Transparent native-to-ascll conversion $\rightarrow$ Create UTF-8 files: 选择 with NO BOM $\rightarrow$ Apply

2.7 主题

2.7.1 代码主题

  • 安装主题

    • 下载地址：http://www.themesmap.com/theme.html?t=down

    • 下载主题：Obsidian

  • 导入主题：打开设置界面 $\rightarrow$ Editor $\rightarrow$ Color Scheme $\rightarrow$ 齿轮设置 $\rightarrow$ Import Scheme... $\rightarrow$ 选择 Obsidian 主题所在路径 $\rightarrow$ OK

  • 设置主题：Color Scheme $\rightarrow$ Python $\rightarrow$ 选择 Obsidian 主题 $\rightarrow$ OK

2.7.2 IDE 主题

（1）方式一

  • 欢迎界面点击 Customize $\rightarrow$ Color theme，选择要设置的主题。

（2）方式二

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ Appearance $\rightarrow$ Theme，选择要设置的主题。

    • 获取更多主题：Get more themes $\rightarrow$ 选择要安装的主题 Install $\rightarrow$ Accept $\rightarrow$ Apply

2.8 设置字体

2.8.1 IDEA 界面字体

（1）方法一

  • 在欢迎界面下，点击 Customize $\rightarrow$ 在 Accessibility 中进行设置 IDEA 界面字体大小。

（2）方法二

  • 进入设置界面 $\rightarrow$ Appearance & Behavior $\rightarrow$ Appearance $\rightarrow$ 设置 IDEA 界面字体大小。

2.8.2 代码编辑界面字体大小

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ Font $\rightarrow$ 设置代码编辑界面字体大小及行距 $\rightarrow$ OK

2.8.3 Ctrl + 鼠标滚轮修改字体大小

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ 勾选 Change font size with Ctrl+Mouse Wheel in: $\rightarrow$ 自选 Active editor 或 All editors $\rightarrow$ OK

2.8.4 控制台字体

  • 默认情况下无需调整，会跟随代码编辑界面使用一致的设置，如需自定义则可以进入设置界面 $\rightarrow$ Editor $\rightarrow$ Color Scheme $\rightarrow$ Console Font $\rightarrow$ 勾选 Use console font instead of the default，然后自定义控制台字体

2.9 行号和方法分隔符

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Appearance $\rightarrow$ 勾选 Show line numbers 显示行号，勾选 Show method separators 显示方法分隔符

2.10 忽略大小写匹配

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Code Completion $\rightarrow$ 取消勾选 Match case $\rightarrow$ OK

2.11 配置自动导包

  • 进入设置界面 $\rightarrow$ Editor $\rightarrow$ General $\rightarrow$ Auto Import $\rightarrow$ 勾选 Show auto-import tooltip 和 from <module> import <name>

2.12 更改 Pycharm 缓存目录

  • 修改 Pycharm 配置文件 idea.properties，该文件位于安装路径 C:\Program Files\JetBrains\PyCharm 2022.3\bin 中。

  • 将文件中的 idea.config.path、idea.system.path、idea.plugins.path 和 idea.log.path 按自己需求进行修改，并且去除注释符号。注：路径中的 \ 要修改为 /。

三、PyCharm 快捷键

  • 修改快捷键 File $\rightarrow$ Settings... $\rightarrow$ Keymap

    • start new line 快捷键修改为 Ctrl + Enter：右键 Remove Shift+Enter $\rightarrow$ 右键 Add Keyboard Shortcut $\rightarrow$ Remove

    • run context configuration 快捷键修改为 Shift + Enter：同理

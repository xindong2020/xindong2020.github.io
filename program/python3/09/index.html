一、函数定义

1.1 函数简介

  • 函数是一段 封装好 的，可重复使用 的，用来实现单一或相关联功能的代码。

  • 函数可以简化程序，清晰结构。

  • 在类中定义的函数又被称为 方法。

1.2 函数定义的语法

  • 语法格式：

    • 关键字：

      • def：定义函数的关键字，所有函数都以此开头，后接函数名。

      • return：结束函数，返回一个值给调用者，不填写返回值的 return 相当于返回 None，即 return None。

      • pass：如果函数不做任何事情，或者函数体中的某些分支语句不做任何事情，可以使用 pass 语句用作占位语句，保持程序结构的完整性。

    • 函数名：一个符合 Python 语法的标识符，建议做到见名知意。

    • 形参列表：形参列表中的参数用逗号 , 分开，声明了调用函数时需要传入的参数。如果函数不接收任何参数，参数列表就为空 ()。

      • 注：函数不需要参数，也要保留括号 “()”，否则 Python 解释器报错 “invaild syntax”。

    • 函数体：定义一个函数功能的所有代码组成的整体，以冒号 : 开头并缩进。第一行语句可以选择性地使用注释书写函数说明。

      • 注：可以使用 pass 定义空函数。

    • 返回值：使用 return 返回函数被调用后得出的结果，可以是表达式，也可以是变量，默认为 return None。return 语句起到结束函数的作用。

      • 注：Python 函数支持多返回值（封包）。

  • 示例 2：打印三角形图案

1.3 文档字符串（DocStrings）

  • 文档字符串是一个重要工具，用于解释说明函数的功能与用法，是一个函数的使用说明书。

  • 我们可以在函数体的 第一行 使用 一对 三个单引号 ''' 或 一对 三个双引号 """ 来定义文档字符串。

  • 文档字符串应该：

    • 包含函数做什么，而不是怎么做的。

    • 让使用者不需要看一行代码，只需要看文档就可以。

    • 包含输入和输出的详细描述。

    • 对于复杂的代码可以在代码旁边加注释。

    • 覆盖基类的方法可以加：See base class

  • 在文档字符串中，应该根据不同的内容进行分组。文档字符串常用的分组：

    • Args：

      • 列出每个参数的名字，并在名字后使用一个冒号和一个空格，分隔对该参数的描述。

      • 如果描述太长超过了单行 80 字符，使用 2 或 4 个空格的悬挂缩进（与文件其他部分保持一致）。

      • 描述应该包括所需的类型和含义，如果一个函数接受 *args（元组可变长参数列表）或 **bar（字典可变长参数列表），应该详细列出 *foo 和 **bar。

    • Returns：（或 Yields：用于生成器）

      • 描述返回值的类型和含义。如果函数返回 None，这一部分可以省略。

    • Raises：

      • 列出与接口有关的所有异常。

  • 通过调用 Python 的 help() 内置函数 或 __doc__ 属性，我们可以查看某个函数的使用说明文档。

    • 示例：help(rectangleArea) 或 rectangleArea.__doc__

1.4 类型标注

1.4.1 概述

  • Python 属于动态类型语言，在创建数据时并不强制标注函数和变量类型，所以只有在运行代码的时候才能够知道变量类型。这通常会让我们在使用他人代码时传入错误的变量类型。为了解决这些问题，在 Python 3.5 中，Python PEP 484 引入了类型注解（Type Hints），在 Python 3.6 中，PEP 526 又进一步引入了变量注解（Variable Annotations）。

    • PEP 是 Python Enhancement Proposals 的缩写。一个 PEP 是一份为 Python 社区提供各种增强功能的技术规格，也是提交新特性，以便让社区指出问题，精确化技术文档的提案。

  • 基本内置类型的表述能力有限，不能说明复杂的类型组成情况，因此 Python 自 3.5 开始发布了 typing 包，为类型标注提供了支持，让我们能够更好的标注变量类型。

  • 类型标注可被用于第三方工具，例：类型检查器、集成开发环境、静态检查器等。

  • Python 对类型标注并 不强制，不按照类型标注传入变量也不会报错，仅会 警告，PyCharm 支持 typing 检查，错误时会出现黄色波浪线警告。

  • 类型标注以 字典 的形式存放在函数的 __annotations__ 属性中。

1.4.2 语法格式

  • 变量类型：在变量名后加一个冒号 :，冒号后加变量的数据类型（例：int、float、list、dict 等）。

  • 函数参数：和普通变量类型的语法格式没区别，都是冒号 + 数据类型。

  • 函数返回类型：在形参列表的括号后加一个箭头 ->，箭头后加返回值的类型，类型标注完成后加函数的冒号 :。

  • 格式要求（PEP 8，非强制）：变量名和冒号之间无空格，冒号和数据类型之间加一个空格，箭头左右均有一个空格。

1.4.3 常用类型标注

1）基本内置类型

  • 常用：int 整型、float 浮点型、bool 布尔型、str 字符串类型。

  • Python 中的内置类型（存放于 builtins 模块 中）都可以用于类型标注。例：int、float、bool、complex、str、list、tuple、dict、set、range、reversed、zip 等。

2）typing 模块提供的类型

  • 除了内置在 builtins 模块中的类型，typing 模块提供了更强大的类型支持。使用 print(dir(typing)) 即可查看。

  • typing 模块中常用的类型：List 列表、Tuple 元组、Dict 字典、Set 集合、Iterable 可迭代类型、Iterator 迭代器类型、Generator 生成器类型。

    • 注：使用 typing 模块中的类型，需要提前导入。例：from typing import List, Tuple, Dict。

（2）Any 类型
  • Any 是一种特殊的类型，它可以表示 所有类型，类型检查器认为所有类型均与 Any 兼容，Python 中所有无明显类型标注的地方，都默认是 Any 类型。例：num = 7 与 num: Any = 7 等价。
（3）Union 类型
  • Union 联合类型的语法格式：Union[X, Y]，相当于 X | Y，即并集，意思是类型是 X 或 Y，效果与关键字 or 等同。例：lst: List[Union[int, float, str]] = [5, 2, 7, "xin", 6.25]
（4）Optional 类型

  • Optional 是指这个参数可以为空或是已经声明的类型，即 Optional[X] 等价于 Union[X, None]。

  • Optional 并不等价于可选参数，当它作为参数类型标注的时候，不意味着这个参数可以不传递了，而是说这个参数可以传为 None。

（5）Sequence 类型
  • Sequence 是 collections.abc.Sequence 的泛型，在某些情况下，我们可能并不需要严格区分一个变量或参数到底是列表 list 类型还是元组 tuple 类型，我们可以使用一个更为泛化的类型，叫做 Sequence。
（6）Callable 类型

  • Callable 表示可调用类型，格式：Callable[[ArgsType], ReturnType]，如果只指定可调用类型的返回值类型，则可以使用 Callable[..., ReturnType] 的形式。

    • Python 中有 7 种可调用对象：内置函数（例：isinstance()）、内置方法（例：list.append()）、自定义函数、生成器函数、类的方法、类本身、类的实例。
（7）Iterator 类型
  • Iterator 表示迭代器类型，迭代器属于可迭代对象，可迭代对象都构建了 __iter__ 方法，迭代器还需构建 __next__ 方法。
（8）Generator 类型

  • Generator 表示生成器类型，Generator[YieldType、SendType、ReturnType]

  • 生成器是一种特殊的迭代器，内部支持了生成器协议，不需要明确定义 __iter__ 方法和 __next__ 方法。

（9）Iterable 类型
  • Iterable 表示可迭代类型，collections.abc.Iterator 的泛型版本，例：字符串、列表、元组、集合、字典等都属于可迭代对象。

二、函数调用

2.1 函数调用

  • 函数调用也就是执行函数体，可以在一个函数中调用其他函数。

  • 函数调用的基本语法格式：[返回值] = 函数名([实参列表])

    • 返回值：如果调用的函数存在返回值，我们可以通过一个变量来接收，也可以不接收。

    • 函数名：要调用的函数的名称。

    • 实参列表：创建函数时要求传入的各个形参的值。实参数量与形参数量要保持一致。

  • 示例：area = rectangleArea(6, 8)

2.2 参数传递

2.2.1 形参 VS 实参

  • 形参（形式参数）：在定义函数时，函数名后面括号中的参数可以看做是一个占位符，它没有数据，只能等到函数被调用时接收传递进来的数据，所以称为形式参数，简称形参。

  • 实参（实际参数）：在调用函数时，函数名后面括号中的参数包含了真实存在的数据，会被函数内部的函数体使用，所以称为实际参数，简称实参。

2.2.2 可变类型 VS 不可变类型

  • 可变类型（可变数据类型）：当该数据类型对应的变量的 值发生变化 时，如果变量的 内存地址不发生改变，那么这个变量对应的数据类型就是 可变数据类型。

    • list（列表）、dict（字典）、set（集合）

  • 不可变类型（不可变数据类型）：当该数据类型对应的变量的 值发生变化 时，如果变量的 内存地址发生改变，那么这个变量对应的数据类型就是 不可变数据类型。

    • numbers （数值类型）、string（字符串）、tuple（元组）。

2.2.3 传递方式

  • 值传递：在调用函数时将实参复制一份传递到函数中，这样在函数中如果对参数进行修改，将不会影响到实参。

  • 引用传递：在调用函数时将实参的地址传递到函数中，那么在函数中对参数所进行的修改，将影响到实参。

    • 函数 的 引用 表示函数存储在内存中的地址。

  • 在 python 中，一切皆对象，数字、字符串、元组、列表、字典、函数、方法、类、模块等都是对象，所以参数传递时，只是让新变量与原变量 指向相同的对象 而已，并不存在值传递或是引用传递一说。准确地说，Python 的参数传递是 赋值传递，或者叫作 对象的引用传递。

    • 如果函数收到的是一个 可变对象（例：列表、字典、集合等）的引用，就能修改对象的原始值，相当于 “引用传递”。

    • 如果函数收到的是一个 不可变对象（例：数字、字符串、元组等）的引用，就不能直接修改原始对象，相当于 “值传递”。

2.2.4 参数类别

（1）位置参数（必需参数）

  • 调用函数时，按照传递的实际参数的 位置 识别对应的形式参数。

    • 在调用函数时，传递的实际参数的数量必须和形式参数的 数量 一致，否则会抛出 TypeError 异常。

    • 在调用函数时，传递的实际参数的位置必须和形式参数的 位置 一致，否则会抛出 TypeError 异常（数据类型不一致）或得到错误的结果。

（2）关键字参数

  • 调用函数时，按照传递的实际参数的 名字 识别对应的形式参数。

    • 通过此方式指定函数实参时，不再需要与形参的位置完全一致，只需 参数名对应 即可。

    • 在调用函数时，可以同时使用位置参数和关键字参数，但是 位置参数必须放在关键字参数之前。

（3）默认参数

  • Python 允许为形式参数设置默认值，即在定义函数时，直接为形式参数指定一个默认值。

三、lambda 表达式（匿名函数）

3.1 概述

  • lambda 表达式，又称匿名函数，常用来表示内部仅包含 1 行表达式的函数。

  • lambda 表达式的语法格式：lambda [arg1[, arg2, ..., argN]] : expression。

    • 定义 lambda 表达式，必须使用 lambda 关键字。

    • 参数 [arg1[, arg2, ..., argN]] 可以有 0 或多个, 但是后面的 expression 只能有一个。

    • lambda 表达式的返回值就是 expression 的结果，而不需要 return 语句。

  • 匿名函数可以在需要函数对象的任何地方使用（例：赋值给变量、作为参数传入其他函数等），因为匿名函数可以作为一个表达式，而不是一个结构化的代码块。

  • 不推荐把 lambda 表达式赋给另一个变量，再通过变量调用函数这种方式。因为 lambda 表达式本就是一个匿名的函数，赋值给变量再调用就失去了匿名的意义。

4.4 max

  • 格式：max(iterable, *[, default=obj, key=func]) 或 max(arg1, arg2, *args[, key])，返回可迭代对象中最大的元素或者返回多个实参中的最大值。

    • 如果 max 函数只传入了一个位置参数，那么使用的是 max(iterable, *[, default=obj, key=func]) 这个函数，因此实参必须是可迭代对象，max 函数返回可迭代对象中最大的元素。

      • 当可迭代对象为空时，如果没有指定 default 默认值，则抛出异常 ValueError: max() arg is an empty sequence。

      • 如果传入了多个位置参数，则使用的是 max(arg1, arg2, *args[, key]) 这个函数，返回多个实参中的最大值。

      • 当传入的参数为数据类型不一致时，传入的所有参数将进行隐式数据类型转换后再比较，如果不能进行隐式数据类型转换，则会报错。

      • 关键字参数 key 可以接收一个函数，用来指定获取最大值的方法。

  • Python 中比较大小通常指的是元素的大小比较：

    • 数字类型（int、float 等）：按 数值 大小比较。

    • 字符串类型（str）：按 字典序 比较，即从左到右依次比较每个字符的 ASCII 码值大小。

    • 序列类型（list、tuple）：从左到右依次比较 每个元素 的大小，如果有一个元素不相等，则返回比较结果，否则继续比较后面的元素。

      • 如果 序列长度 不同，而且短序列和长序列的前几项对应相等，则 默认长序列大于短序列，与长序列剩余项的值无关。

    • 映射类型（dict）：无法进行大小比较。

4.6 pow

  • 格式：pow(base, exp[, mod])，返回一个数 base 的 exp 次幂，即 base ** exp。如果指定了 mod 参数，则先进行幂运算，然后对 mod 取模，即 (base ** exp) % mod。

    • base 和 exp 可以为 int 或 float 类型，但是 mod 必须为 int 类型。

4.7 round

  • 格式：round(number[, ndigits])，返回 number 舍入到小数点后 ndigits 位精度的值，通常用于 float 类型数据，保留指定位数。

    • ndigits 只能为 int 类型，如果 ndigits 被省略或为 None，则返回最接近输入值的整数。例：round(2.675) 返回的是 3，round(3.14) 返回的也是 3。

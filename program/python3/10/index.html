一、封包、解包

1.1 封包（pack）

  • 封包就是将多个元素合起来作为一个整体使用。

  • 将多个值赋值给一个变量时，Python 会自动将这些值封装成元组，这个特性称之为 封包。例：

  • 当函数的不定长参数接收多个参数值时，Python 会自动进行封包。例：

  • 当函数返回多个值时，Python 会自动进行封包。例：

  • 实际开发过程中，很少主动进行封包操作。

1.2 解包（unpack）

  • 解包就是将容器里面的元素逐个取出来使用。

  • 可迭代对象都支持解包。

    • 可迭代对象：列表、元组、字典、集合、字符串、range、迭代器（reversed、zip、enumerate）等

1.2.1 赋值多变量

（1）可迭代元素数量 = 变量个数

  • 将一个可迭代对象赋值给多个变量时，Python 会自动解包，将可迭代对象中的元素逐一赋值给每个变量，因此变量个数要与可迭代对象的元素数量相同，否则抛出异常 ValueError: too many values to unpack 或 ValueError: not enough values to unpack。

  • 示例：

（2）可迭代元素数量 > 变量个数

  • 当我们就想将元素数量大于变量个数的可迭代对象赋值给变量时，可以在其中一个变量前面加一个星号（*），代表这个变量可接收 0 个、1 个或多个元素，并把这些值组成一个 列表。

  • 多个变量中不允许存在多个带星号（*）的变量。

  • 解包过程：首先将其他变量根据位置确定对应赋值的元素，剩下的元素组成一个 列表 赋值给带星号（*）的变量。

  • 带星号（*）的变量与不定长参数 *args 不同：

    • * 变量是不贪婪的，会先让其他变量赋值，剩余的值全部给* 变量，* 变量后面的变量能够被赋值。

    • 不定长参数 *args 是贪婪的，会将所有的实参值全部获取，*args 后面的参数不会被赋值。

  • 当仅为一个带星号（*）的变量赋值可迭代对象时，必须要在变量名后添加 逗号（,）。例：*a, = [10]。

  • 示例：

（3）接收函数返回值

  • 当函数返回多个值时，Python 会自动将多个值封包成一个元组，使用多个变量接收函数返回值时，Python 会自动解包，将元组中的元素逐一赋值给每个变量。

  • 示例：

1.2.2 交换两个变量值

  • 在 Python 中，交换两个变量的值非常方便，本质上也是自动解包的过程。

  • 示例：

    • b, a = a, b 将 a, b 封包成为一个元组，然后赋值给 b, a 进行解包。

二、命名空间与作用域

2.1 命名空间

2.1.1 概述

  • 命名空间（Namespace）是从名称到对象的映射，大部分命名空间是通过字典来实现的（内置命名空间不是由字典实现，而是由 builtins 模块实现）。

  • 作用：命名空间提供了一种在项目中避免名字冲突的方法。各个命名空间是独立的，没有任何关系的，所以同一个命名空间中不能有重名，但不同的命名空间可以重名且相互没有任何影响。

2.1.2 分类

  • Python 命名空间可分为 3 类，不同的命名空间也具有不同的生命周期。

（1）内置命名空间（built-in names）

  • 内置命名空间是 Python 自带的命名空间，任何模块都可以访问，包含了像 abs() 这样的内置函数（function）和 BaseException 这样的内置异常（exception）名等。

    • builtins 模块就是内置命名空间。使用下列代码即可查看 builtins 模块中的属性、方法、类型组成的列表。

  • 生命周期：在 Python 解释器启动的时候被创建，在解释器终止的时候被删除。

（2）全局命名空间（global names）

  • 全局命名空间记录了 模块 中定义的对象，包含模块中定义的函数、类、其他导入的模块、模块级的变量与常量。

    • 解释器会为程序中 import 加载的模块创建一个全局命名空间。

  • globals() 内置函数会以字典形式返回 当前模块 的 全局命名空间。

  • 生命周期：一个模块的全局命名空间在这个模块被 import 的时候创建，并且模块的全局命名空间会一直存在，直到解释器终止。

（3）局部命名空间（local names）

  • 局部命名空间，也称为本地命名空间，是每个 函数 所拥有的命名空间，记录了函数中定义的所有变量，包括函数的入参、内部定义的局部变量。

  • locals() 内置函数会以字典形式返回 当前函数 的 局部命名空间。该函数需要在函数内部调用。

  • 生命周期：局部命名空间在一个函数被调用的时候创建。如果这个函数 return 或 raise 了一个不由该函数处理的异常，则该命名空间被删除。注：每个递归调用都有它们自己的局部命名空间。

2.1.3 命名空间查找、创建、销毁顺序

（1）命名空间查找顺序

  • 如果程序执行时使用一个变量 var，那么 Python 查找变量的顺序为：

    $$局部的命名空间 \rightarrow 全局命名空间 \rightarrow 内置命名空间$$

    如果按照这个顺序找不到相应的变量，Python 将放弃查找并抛出异常：NameError: name 'var' is not defined。

（2）命名空间创建顺序
  • Python 解释器启动 $\rightarrow$ 创建内置命名空间 $\rightarrow$ 加载模块 $\rightarrow$ 创建全局命名空间 $\rightarrow$ 函数被调用 $\rightarrow$ 创建局部命名空间。
（3）命名空间销毁顺序
  • 函数调用结束 $\rightarrow$ 销毁函数对应的局部命名空间 $\rightarrow$ Python 解释器终止 $\rightarrow$ 销毁全局命名空间 $\rightarrow$ 销毁内置命名空间。

2.1.4 相关内置函数

（1）eval

  • 格式：eval(expression[, globals[, locals]])，用于执行一个字符串形式的表达式 expression，并 返回执行结果。

    • expression：字符串，内容为表达式。

    • globals：字典，默认使用当前作用域的全局命名空间。

    • locals：字典，默认使用当前作用域的局部命名空间。

      • 如果提供了 globals 参数，而 locals 参数被忽略，那么它将会取与 globals 相同的值。

  • 如果 globals 与 locals 参数都为空，那么将遵循命名空间的查找顺
    序查找是否存在变量；如果提供了 globals 参数，而 locals 参数为空，则仅查找 globals 参数中是否存在变量；如果两个参数都不为空，先查找 locals 参数，再查找 globals 参数。如果在指定的命名空间中没有查找到相关变量，会抛出异常 NameError: name 'xxx' is not defined。

  • 示例：

（2）exec

  • 格式：exec(object[, globals[, locals]])，传入 exec 函数的 object 实参可以是 字符串，也可以是 字节码对象。通常 exec 函数配合 compile 函数来使用。无返回值。

    • object 可以是字符串或者字节码对象。

      • 如果 object 实参是字符串则会被 exec 函数编译并执行。

      • 如果是字节码对象则会被直接执行。

    • globals：字典，默认使用当前作用域的全局命名空间。

    • locals：字典，默认使用当前作用域的局部命名空间。

  • 如果 globals 与 locals 参数都为空，那么将遵循命名空间的查找顺
    序查找是否存在变量；如果提供了 globals 参数，而 locals 参数为空，则仅查找 globals 参数中是否存在变量；如果两个参数都不为空，先查找 locals 参数，再查找 globals 参数。如果在指定的命名空间中没有查找到相关变量，会抛出异常 NameError: name 'xxx' is not defined。

（3）eval 和 exec 区别

  • eval 函数返回表达式执行后的结果，而 exec 函数没有返回值。

2.2 作用域

2.2.1 概述

  • Python 作用域就是一个程序可以直接访问命名空间的正文区域。

  • Python 程序中，直接访问一个变量，会从内到外依次访问所有的作用域直到找到，否则会报未定义的错误。

  • 变量并不是在哪个位置都可以访问的，访问权限取决于这个变量是在哪里赋值的，也就是在哪个作用域内的。

  • 变量的作用域决定了在哪一部分程序可以访问哪个特定的变量名称。

  • 循环语句、条件语句都不会引入新的作用域，只有 模块、类、函数（def、lambda）和 推导式 会引入新的作用域。例：

2.2.2 分类

  • Python 的作用域一共有 4 种（LEGB 作用域）
（1）L（Local）局部作用域

  • 最内层，包含局部变量，例：函数/方法内部。

  • 在函数内部创建的变量属于该函数的局部作用域，并且只能在该函数内部使用。

  • 默认情况下局部是无法修改全局变量的。例：

（2）E（Enclosing）闭包函数外函数作用域

  • 在一个外函数中定义了一个内函数，内函数中调用了外函数中定义的变量，并且外函数的返回值是内函数的引用，这样就构成了一个闭包。例：

    • name = "dong" 和 return inner 的作用域就是 Enclosing 作用域。
（3）G（Global）全局作用域

  • 模块文件中声明的变量，即模块级别的变量，具有全局作用域。

  • 模块的全局变量就是一个模块对象的属性，仅限于单个模块文件中。

（4）B （Built-in） 内置作用域
  • 系统内解释器定义的变量，例：预定义在 builtin 模块内的变量（定义在模块内、类和函数之外）。

2.2.3 搜索顺序

  • Python 以 L $\rightarrow$ E $\rightarrow$ G $\rightarrow$ B 的搜索顺序查找变量，即：局部作用域搜索不到，便会去闭包函数外的函数中搜索，再搜索不到就会去全局作用域搜索，最后搭配内置作用域中搜索。如果所有作用域都搜索不到，就会抛出异常 NameError: name 'xxx' is not defined。

  • Python 只会从当前作用域开始搜索，而且不会向更深层搜索。

  • 示例：

2.2.4 全局变量和局部变量

  • 定义在函数内部的变量拥有一个局部作用域（Local），被叫做局部变量。

  • 闭包函数外函数（Enclosing）包含了非局部（non-local）也非全局（non-global）的变量。

  • 定义在函数外的拥有全局作用域（Global）的变量，被称为全局变量。

（1）global

  • 在普通函数中，虽然可以直接使用全局变量，但是不可以直接修改全局变量。从变量的作用域来说，一旦修改全局变量，那么 Python 就会创建并使用一个同名的局部变量。因此，如果需要在普通函数中修改全局变量，需要使用 global 关键字指定当前变量使用的是外部的全局变量。

三、常用高阶函数

  • 高阶函数就是参数或（和）返回值为函数的函数。

3.1 filter

  • 格式：filter(function, iterable)，Python 内置函数，它使用函数 function 对可迭代对象 iterable 中的数据进行过滤，将 iterable 中的每个元素作为参数传递给 function，根据返回结果进行判断，将判断为 True 的元素构建成新的迭代器并返回。

  • function 是必需参数，可以为仅接收一个实参的函数或 None。

  • 如果 function 为 None，直接判断 iterable 中的元素，将判断为 True 的元素构建成新的迭代器并返回。等价于 filter(bool, iterable)，即 function 设置成内置函数 bool()。

  • filter(function, iterable) 相当于一个生成器表达式，当 function 参数不为 None 时，等价于 (item for item in iterable if function(item))，当 function 为 None 时，等价于 (item for item in iterable if item)。

  • 示例：

3.2 map

  • 格式：map(function, *iterables)，Python 内置函数，该函数将可迭代对象 iterable 中每一项作为参数调用函数 function，返回包含每次 function 函数返回值的迭代器。

    • function：函数，可以是 lambda 函数、内置函数、自定义函数等，参数个数要与 iterable 相同。

    • *iterables：可迭代对象，不定长参数，当有多个可迭代对象时，最短的可迭代对象耗尽则整个迭代就结束。

  • 示例：

3.3 reduce

  • 格式：reduce(function, iterable[, initial])。

    • function：必需参数，要求是能够接收两个实参的函数。

    • iterable：必需参数，可迭代对象。

    • initial：可选参数，初始值。

  • reduce 函数的执行流程是 ：在迭代 iterable 的过程中，首先把前两个元素（只能两个）传给函数，然后把函数执行得到的结果和第三个元素作为两个实参传给函数，函数执行后得到的结果又和第四个元素作为两个实参传给函数，依此类推，最终 返回一个函数计算结果。

    • 当指定 initial 参数时，先把 initial 值和 iterable 的第一个元素作为两个实参传给函数。

    • 如果 iterable 为空，则函数返回 initial，如果 initial 没有指定则抛出异常 TypeError: reduce() of empty sequence with no initial value。

    • 如果 iterable 只有一个元素且没有指定 initial，则返回该元素。

  • 在 Python2 中 reduce 函数是内置函数，而在 Python3 中 reduce 函数被放置到 functools 模块中，所以在使用的时候需要引入 functools 模块来调用 reduce 函数：from functools import reduce。

前言

  • 字典（<class 'dict'>）和 集合（<class 'set'>）不是序列，但是可迭代对象，可以使用 collections 库中的 abc 模块里面的 Sequence 基类 来判断一个类型是不是属于序列。

  • 集合和字典不支持索引、切片、相加和相乘操作。

一、dict 字典

1.1 概述

  • 类型 <class 'dict'>。

  • 字典类型是 Python 中唯一的 映射类型，即通过一个元素，可以唯一找到另一个元素。

  • 字典中，习惯将各元素对应的 索引 称为 键（key），各个键对应的 元素 称为 值（value），键及其关联的值称为 “键值对”。

  • 字典的每个键值对使用冒号 : 隔开写成 key: value 的格式，每个键值对
    之间用逗号 , 隔开，包括在花括号 {} 中。例：{"name": "dong", "age": 18}。

  • 字典是一种 可变数据类型。

  • 字典作为可迭代对象时，只有键参与迭代。例：

  • 字典中，键 必须 唯一，但值可不唯一，即为不同键赋同一值。

    • 如果字典的键重复，会自动去重，即键保持不变，留下最新的值，相当于更新键值对。

  • 字典中，值可以取任何数据类型，但 键 必须是 不可变类型 且内部不包含可变数据类型（例：元组中包含列表，这种也是不允许的）。

1.2 创建字典的六种方式

1.2.1 使用 { } 创建字典

（1）直接在空字典 {} 里面写键值对

  • 由于字典中每个元素都包含 两部分，分别是 键（key）和 值（value），因此在创建字典时，键和值之间使用 冒号 : 分隔，相邻元素之间使用 逗号 , 分隔，所有元素放在大括号 {} 中。

  • 格式：dictname = {'key1': 'value1', 'key2': 'value2', ..., 'keyN': valueN}，其中 dictname 表示字典变量名，'keyN': valueN 表示各个元素的键值对。

1.2.2 使用 dict 内置函数创建字典

（1）方式一：使用关键字参数创建

  • 格式：dict(**kwarg) 用于创建字典。当不传递 **kwarg 参数时，创建一个空字典，即 dict() 创建空字典。

    • **kwarg 是 双星号 参数，即 字典可变长 参数，表示 dict 函数可以接受 0个或多个参数，并且这些传进来的参数要求以 关键字参数 的形式导入，且都被存放在字典 kwarg 中。

    • 双星号（**）参数与单星号（*）参数都是 不定长参数。

    • 关键字参数中的 关键字 会自动变成 字符串 类型，作为字典中的 键。

  • 示例：

    • 使用此方式创建字典时，字符串键不能带引号。
（2）方式二：使用映射对象创建字典

  • 格式：dict(mapping, **kwarg) 用于创建字典。mapping 是一种关联式的容器类型，它存储了对象与对象之间的映射关系。

  • 通过 zip 函数、map 函数等可以直接生成 mapping 类型的数据。

  • 示例：

（3）方式三：使用可迭代对象创建字典

  • 格式：dict(iterable, **kwarg) 用于创建字典。iterable 数据中每一项的长度必须为 2，其中第 1 个元素作为键，第 2 个元素作为值。否则抛出异常：ValueError: dictionary update sequence element #0 has length 3; 2 is required。

  • 示例：

1.2.3 使用类方法 fromkeys() 创建字典

  • 格式：dictname = dict.fromkeys(iterable[，value=None])，用于创建带有默认值的字典。其中 iterable 参数表示字典中的所有键，value 参数表示默认值，如果不写，则为空值 None。这种创建方式通常用于 初始化字典。

  • fromkeys 是一种 类方法，直接使用字典类型 dict 进行调用。

二、set 集合

2.1 概述

  • 类型 <class 'set'>。

  • Python 中的集合用来保存 不重复 的元素，即集合中的元素都是 唯一 的，互不相同。

  • Python 中的集合是 无序 的，所以每次输出时元素的排序顺序可能会不同。

  • 集合不是序列，但是可迭代对象，是 可变 的。注：这里的可变不是指元素可变。

  • 从 形式 上看，集合与字典类似，Python 集合会将所有元素放在一对大括号 {} 中，相邻元素之间用 “,” 分隔：{element1, element2, ..., elementn}。

  • 从 内容 上看，同一集合中，只能存储 不可变 的数据类型（数值、字符串、元组等），无法存储可变的数据类型（列表、元组套列表、字典、集合等），否则 Python 解释器会抛出 TypeError 异常。

  • Python 中有 两种集合 类型，一种是 set 类型 的集合，另一种是 frozenset 类型 的集合，它们唯一的区别是：set 类型集合可以进行添加、删除元素的操作，而 forzenset 类型集合不可以。因此，set 集合是可变的，forzenset 集合是不可变的。

    • 注：不能使用 { } 创建 空集合，因为 Python 解释器会将 { } 视为一个空字典。
（2）使用 set() 函数创建 set 集合

  • 格式：set([iterable])，用于创建集合并返回，其中 iterable 表示字符串、列表、元组、range 对象等可迭代对象。如果未指定 iterable，则返回空集合，即 set() 返回空集合。

    • 注：只能使用 set() 函数创建 空集合。

（15）symmetric_difference_update() 方法

  • 格式：setname.symmetric_difference_update(iterable)，用于将 iterable 中的元素插入到 setname 中，然后再删除 setname 与 iterable 的交集。

    • symmetric_difference_update 方法与 symmetric_difference 方法的区别：symmetric_difference 方法返回一个 新集合，而 symmetric_difference_update 方法是在 原集合 上插入可迭代对象中的元素，然后移除两个对象的交集。

  • 格式：setname.union(*iterable)，返回一个新集合，新集合是两个或多个可迭代对象的 并集，重复的元素只会出现一次。

（17）update() 方法

  • 格式：setname.update(*iterable)，用于向 set 集合中添加可迭代对象中的所有元素，无返回值。

    • 注：使用 update() 方法只能添加 字符串、列表、元组、字典 这类可迭代的数据，不能添加 数字 或 布尔类型（True 和 False）这类不可迭代的数据，否则 Python 解释器会抛出 TypeError 异常。如果添加的元素在集合中已存在，则该元素只会出现一次，重复的会忽略。

2.3 frozenset 类型的集合（set 集合的不可变版本）

  • 格式：frozenset([iterable])，用于创建一个新的 frozenset 类型集合，即 冻结集合。其中 iterable 表示字符串、列表、元组、range 对象等可迭代对象。如果未指定 iterable，则返回空的冻结集合，即 frozenset() 返回空的冻结集合。

  • set 中的元素必须是不可变类型的，而 frozenset 是可以作为 set 元素的。例：

  • set 集合中所有能修改集合的方法（remove()、discard()、add() 等），frozenset 都不支持；set 集合中不修改集合的方法 fronzenset 都支持。

  • 可以通过 dir(frozenset) 来查看 frozenset 集合支持的方法：

    • 不以双下划线 __ 开头结尾的方法都是 frozenset 集合的对象方法，即 copy, difference, intersection, isdisjoint, issubset, issuperset, symmetric_difference, union。

    • frozenset 集合的这些方法和 set 集合中同名方法的功能是一样的。

  • 两种情况下可以使用 fronzenset：

    • 当集合的元素不需要改变时，我们可以使用 fronzenset 替代 set，这样更加安全。

    • 程序要求必须是不可变对象时也可以使用 fronzenset 替代 set。例：字典（dict）的键（key）就要求是不可变对象。set 集合中的元素要求是不可变的，可以使用 fronzenset 集合作为 set 集合中的元素。

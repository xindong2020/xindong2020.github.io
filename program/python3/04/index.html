一、字符串

1.1 概述

  • 类型 <class 'str'>。

  • 字符串是一种 不可变 的数据类型。

  • 字符串也是一种常见的序列，所有标准的序列操作（索引、切片、相加、相乘、in 操作、序列内置函数 等）对字符串同样使用。

  • Python 中没有 字符（char） 这个类型，而是用 “长度为 1 的字符串” 来表示字符的概念。

1.2 字符串拼接

  • 使用加号 + 拼接，相邻的两个字符串之间 不会自动添加空格，

1.3 字符串方法

  • Python 集合的数据类型为 str，我们可使用 dir(str) 来查看该类型包含哪些方法。

  • 因为字符串是 不可变 数据类型，所以字符串方法基本都 具有返回值，即返回一个 新字符串，原字符串并不被修改。

    • 对原字符串的操作其实是应用于原字符串的 拷贝副本，操作完成后再将拷贝副本的地址赋值给新的变量，这种操作又被称为 新建操作。

1.3.28 maketrans() 方法

  • 格式：str.maketrans(x[, y[, z]])，返回一个可供 str.translate() 使用的转换对照表。

    • 如果只有一个参数，则它必须是一个将 Unicode 码位序号（整数）或字符（长度为 1 的字符串）映射到 Unicode 码位序号、字符串（任意长度）或 None 的 字典。例：str.maketrans({"H":"h"})，将 “H“ 映射到 “h“。

    • 如果有两个参数，则它们必须是两个长度相等的字符串。例：str.maketrans("HP","hp")，将 “HP“ 字符串中的每个字符 “H，P“ 分别映射 到 “h，p“。

    • 如果有第三个参数，它必须是一个字符串，其中的每个字符将在结果中被映射到 None。

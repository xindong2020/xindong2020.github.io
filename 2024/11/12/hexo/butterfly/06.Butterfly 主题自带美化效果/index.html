十七、加载动画

  • 将 https://codebyzach.github.io/pace/ 中选定的主题下载到 \themes\butterfly\source\css。

  • pace_css_url 填写 css 文件在渲染后 public 中的地址 /css/center-atom.css。

十九、大图查看模式

  • medium-zoom 或 fancybox 只能开启一个，推荐 fancybox。

  • 如果不想为某张图片添加大图查看模式，可以在 img 标签中添加 class 属性，值为 no-lightbox。例：<img src="xxxx.jpg" class="no-lightbox">。

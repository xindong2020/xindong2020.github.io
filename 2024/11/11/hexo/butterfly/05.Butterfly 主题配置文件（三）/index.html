二、搜索

  • butterfly 主題支持三种搜索方式（algolia_search / local_search / docsearch），本站使用 Algolia 搜索。

2.1 Algolia 搜索

2.1.1 注册账号

  • Algolia官网（ https://www.algolia.com/ ）

  • 在项目根目录执行以下代码下载第三方包：

  • 在 Algolia 注册页面 使用 Google账号、Github账号或邮箱直接进行注册。

  • 登录后进入控制台，按以下步骤创建 index，自定义 indexName。

  • 在控制台进入设置页面，点击 API Keys 获取 Application ID、Search API Key、Admin API Key

  • 在 _config.yml 全局配置文件末尾加入以下内容：

    • content:strip:truncate,0,500

      • content: 文章的内容。

      • strip: 去掉 HTML 标签和样式。

      • truncate,0,500: 截取内容的前500个字符。

    • excerpt:strip

      • excerpt: 文章的摘要。

      • strip: 去掉 HTML 标签和样式。

    • gallery: 文章的图片集字段，不做处理直接索引。

    • permalink: 文章的永久链接，不做处理直接索引。

    • slug: 文章的短链接，不做处理直接索引。

    • title: 文章的标题，不做处理直接索引。

  • 把主题配置文件 _config.butterfly.yml 中 search 的 use 设置为 algolia_search。

三、分享系统

  • butterfly 主题支持两种分享方式，通过设置 use，选择分享方式，可选 sharejs 或 addtoany，如果不提供分享功能则留空。

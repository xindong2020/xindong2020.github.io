Eclipse 的配置及使用

一、配置 Eclipse

1.1 Eclipse 的安装

  • 下载 Eclipse 的压缩包：https://www.eclipse.org/downloads/packages/，将 Eclipse 解压到没有中文且没有空格的路径下。

1.2 设置 workspace

  • 首次启动，选择指定的工作空间（workspace），用于存放 Java 代码 。其中红框处 不建议 选中，如果选中，每次启动 Eclipse 会自动进入选中的 workspace，不利于 Eclipse 开启其他 workspace。

    • 解决勾选上述红框，再次启动 Eclipse 时，不再显示可选择的 Workspace 的问题。
      菜单栏 Window $\rightarrow$ Preferences $\rightarrow$ General $\rightarrow$ Startup and Shutdown $\rightarrow$ Workspace $\rightarrow$ 勾选 Prompt for workspace on startup $\rightarrow$ Apply and Close

  • 点击 Launch 进入主界面，首次进入显示如下。可以去掉 Welcome 页面右下角的选中 ，避免每次启动都显示。

1.3 设置透视图（perspective）

  • 进入 Eclipse 编辑界面后，首先点击右上角红框中的按钮。

    • 也可以 Window $\rightarrow$ Perspective $\rightarrow$ Open Perspective $\rightarrow$ Other...

  • 设置透视图，选择 JavaEE 透视图。

1.4 添加透视图需要显示的结构

  • 点击右上角红框中的搜索按钮。

1.4.1 结构：Package Explorer

1.4.2 结构：Navigator

1.4.3 结构：Outline

  • 将结构 Package Explorer、Navigator 和 Outline 拖拽到 Eclipse 编辑界面左侧。

1.4.4 结构：Console

  • 放到 Eclipse 编辑界面下部左侧

1.5 关闭其它不需要的结构

  • 关闭结构 Project Explorer。

  • 关闭结构 Markers、Properties、Data Source Explorer 和 Snippets。

1.6 设置编码集

  • Window $\rightarrow$ Preferences $\rightarrow$ General $\rightarrow$ Workspace $\rightarrow$ Other $\rightarrow$ UTF-8 $\rightarrow$ Apply and Close

1.7 设置字体、字形、大小

  • Window $\rightarrow$ Preferences $\rightarrow$ General $\rightarrow$ Appearance $\rightarrow$ Colors and Fonts $\rightarrow$ Basic $\rightarrow$ 双击 Text Font 根据个人习惯设置字体、字形、大小 $\rightarrow$ Apply and Close

1.8 设置 package explorer 中右键：new 下显示的结构

  • Window $\rightarrow$ Perspective $\rightarrow$ Customize Perspective... $\rightarrow$ Menu Visibility $\rightarrow$ 从 New 的子列表中选择常用的结构 $\rightarrow$ Apply and Close

1.9 保存当前透视图

  • Window $\rightarrow$ Perspective $\rightarrow$ Save Perspective As... $\rightarrow$ save $\rightarrow$ Yes（覆盖默认的透视图即可）

$\quad$ 注：我们对Eclipse所做的设置，只对当前使用的workspace有效。有关设置的数据保存在workspace文件夹的.metadata子文件夹中。如果创建了新的workspace，则需要重新设置。

二、使用 Eclipse

2.1 创建工程

2.1.1 Create a Java project

  • 填写项目名 Project name $\rightarrow$ 选中 Use default JRE 'jre' and workspace compiler preferences $\rightarrow$ Configure JREs $\rightarrow$ 取消勾选 Create module-info.java file $\rightarrow$ Finish

（1）配置 JRE

  • 增加 JRE 配置环境点击 Add...

  • 选择安装 JRE 的类型：双击 Standard VM

  • 配置 JRE：JRE home 选择 JRE 所在文件夹 $\rightarrow$ Finish

  • 勾选新配置的 JRE。

  • 使得编译器版本与 JRE 版本相符。

    • Compiler $\rightarrow$ Compiler compliance level，选择与 JRE 版本号相同的版本 $\rightarrow$ Apply $\rightarrow$ Apply and Close

（2）配置 Perspective

  • 勾选 Remember my decision $\rightarrow$ No，表示以后都是使用JavaEE的透视图打开java工程。

  • 误选 Open Perspective 恢复：

    Window $\rightarrow$ Preferences $\rightarrow$ General $\rightarrow$ Perspectives $\rightarrow$ 选 In the same window $\rightarrow$ 选 Never open $\rightarrow$ Apply and Close

2.2 创建包

  • src 上右键 $\rightarrow$ New $\rightarrow$ Package

  • 填写包名 $\rightarrow$ 取消勾选 Create package-info.java $\rightarrow$ Finish

2.3 创建类

  • 包名 com.dong.java 上右键 $\rightarrow$ New $\rightarrow$ Class

  • 填写类名 $\rightarrow$ 勾选 public static void main(String[] args) 自动创建 main 方法 $\rightarrow$ Finish

2.4 编写代码并运行

  • HelloWorld.java 代码

  • 运行代码

    • 方法 $1$：点击菜单栏中的运行按钮。

    • 方法 $2$：代码页上右键 $\rightarrow$ HelloWorld.java

三、Eclipse 导入与删除项目

3.1 Eclipse 导入项目

  • File $\rightarrow$ Import...

  • General $\rightarrow$ Existing Projects into Workspace $\rightarrow$ Next

  • 点击 Browse... 选择项目所在路径 $\rightarrow$ 勾选 Project 中的项目 $\rightarrow$ 勾选 Copy projects into workspace $\rightarrow$ Finish

3.2 Eclipse 导入源文件

  • 直接复制（ctrl + c）源文件，然后在指定工程的包下粘贴（ctrl + v）即可 。

3.3 Eclipse 删除项目

  • 项目上右键 $\rightarrow$ Delete

  • 勾选 Delete project contents on disk (cannot be undone)（如果仅想在 workspace 中删除项目，而不想在磁盘中永久删除项目，可不勾选）$\rightarrow$ OK

3.4 Eclipse 关闭项目

  • 项目右键 $\rightarrow$ Close Project

  • 过滤掉已关闭的项目：点击 workspace 右上角 $\rightarrow$ 选择 Filters $\rightarrow$ 勾选 Closed projects $\rightarrow$ OK

四、常用快捷键

  • 补全代码的声明：alt + /

    • 输入时自动补全：Window $\rightarrow$ Preferences $\rightarrow$ Java $\rightarrow$ Editor $\rightarrow$ Content Assist $\rightarrow$ Auto activation triggers for java $\rightarrow$ 在 “.” 后添加 $abc\cdots xyz$ 等 $26$ 个字母 $\rightarrow$ Apply $\rightarrow$ Apply and Close

    • 取消空格键 “键入自动提示” 的功能：Window $\rightarrow$ Preferences $\rightarrow$ Java $\rightarrow$ Editor $\rightarrow$ Content Assist $\rightarrow$ 勾选 Disable insertion triggers except 'Enter' $\rightarrow$ Apply $\rightarrow$ Apply and Close

  • 快速修复：ctrl + 1

  • 批量导包：ctrl + shift + o

  • 使用单行注释：ctrl + /

  • 使用多行注释：ctrl + shift + /

  • 取消多行注释：ctrl + shift + \

  • 复制指定行的代码：ctrl + alt + down 或 ctrl + alt + up

  • 删除指定行的代码：ctrl + d

  • 上下移动代码：alt + up 或 alt + down

  • 切换到下一行代码空位：shift + enter

  • 切换到上一行代码空位：ctrl + shift + enter

  • 如何查看源码：ctrl + 选中指定的结构 或 ctrl + shift + t

  • 退回到前一个编辑的页面：alt + left

  • 进入到下一个编辑的页面（针对于上面那条来说）：alt + right

  • 光标选中指定的类，查看继承树结构： ctrl + t

  • 复制：ctrl + c

  • 撤销：ctrl + z

  • 反撤销：ctrl + y

  • 剪切：ctrl + x

  • 粘贴：ctrl + v

  • 保存：ctrl + s

  • 全选：ctrl + a

  • 格式化代码：ctrl + shift + f

  • 选中数行，整体向右移动：tab

  • 选中数行，整体向左移动：shift + tab

  • 在当前类中，显示类结构，并支持搜索指定的方法、属性等：ctrl + o

  • 批量修改指定的变量名、方法名、类名等：alt + shift + r

  • 选中的结构的大小写的切换，变成大写：ctrl + shift + x

  • 选中的结构的大小写的切换，变成小写：ctrl + shift + y

  • 调出生成 getter / setter / 构造器 等结构：alt + shift + s

  • 显示当前选择资源（工程 $or$ 文件）的属性：alt + enter

  • 快速查找（参照选中的 $word$ 快速定位到下一个位置）：ctrl + k

  • 关闭当前窗口：ctrl + w

  • 关闭所有的窗口：ctrl + shift + w

  • 查看指定的结构使用过的地方：ctrl + alt + g

  • 查找与替换：ctrl + f

  • 最大化当前的 View：ctrl + m

  • 直接定位到当前行的首位：home

  • 直接定位到当前行的末位：end

  • 打开声明该引用的文件：F3

  • 调试执行：

    • 打开类型层次结构：F4

    • 单步跳入：F5

    • 单步跳过：F6

    • 单步跳出：F7

    • 恢复正常执行（直到遇到下一个断点）：F8
